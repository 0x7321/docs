## 服務區域

北極熊實驗室（bearlab.ai）面向**美國與歐盟**司法管轄區直接提供服務。

亞太區域由**區域合作伙伴**提供服務：該區域的 API 請求須使用專用介面域名 **`https://bearlab.in`**，而非 `bearlab.ai`。該區域服務的相關政策由區域合作伙伴自行制定。亞太區域使用者請自行確保符合所在地的法律法規。

| 區域 | 介面地址 |
| --- | --- |
| 美國 / 歐盟 | `https://bearlab.ai/v1` · `https://bearlab.ai/anthropic` |
| 亞太區域 | `https://bearlab.in/v1` · `https://bearlab.in/anthropic` |

## Service Availability

BearLab (bearlab.ai) provides services directly in the **United States and European Union** jurisdictions.

The Asia-Pacific region is served by a **Regional Partner**. API requests from this region must use the dedicated endpoint **`https://bearlab.in`** instead of `bearlab.ai`. Policies for service in this region are set by the Regional Partner. Users in the Asia-Pacific region are responsible for compliance with applicable local laws.

| Region | Endpoint |
| --- | --- |
| US / EU | `https://bearlab.ai/v1` · `https://bearlab.ai/anthropic` |
| Asia-Pacific | `https://bearlab.in/v1` · `https://bearlab.in/anthropic` |
