[Cline](https://github.com/cline/cline) 及其增強分支 [Roo Code](https://github.com/RooVetGit/Roo-Code) 都是 VS Code 裡的 AI 程式設計擴充套件，透過 OpenAI 相容介面接入。

## 配置

<Steps>
  <Step title="選擇 Provider">
    在擴充套件設定中，**API Provider** 選擇 **OpenAI Compatible**。
  </Step>
  <Step title="填寫連線資訊">
    - **Base URL**：`https://bearlab.ai/v1`
    - **API Key**：你的 `sk-...` 金鑰
    - **Model ID**：要使用的模型，如 `claude-sonnet-4-5` 或 `gpt-4o`
  </Step>
</Steps>

<Note>
  選 **OpenAI Compatible** 而不是 Anthropic，Base URL 用 `/v1`。程式設計類任務建議選上下文視窗較大、程式碼能力強的模型。Model ID 必須是 [模型介面](/zh-Hant/guides/models) 中真實存在的 ID。
</Note>
