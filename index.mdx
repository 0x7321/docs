---

BearLab AI (bearlab.ai) 將來自數十個提供商的對話模型整合到一個 API 中。建立一個密鑰，設定一個 `base_url`，即可呼叫來自 OpenAI、Anthropic、Google、DeepSeek 及其他廠商的最新模型，無需分別向各家廠商註冊和充值。

該介面相容於 OpenAI 和 Anthropic 協定，因此現有程式碼通常只需變更一行 `base_url` 就能使用。

## 區域 base URL

選擇您所在地區的 base URL，在兩者之間使用相同的 API 密鑰。

| 地區 | API 網域 | OpenAI 相容 | Anthropic 相容 |
| --- | --- | --- | --- |
| 美國 / 歐洲 | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| 亞太地區 | `https://bearlab.space` | `https://bearlab.space/v1/chat/completions` | `https://bearlab.space/v1/messages` |

<Note>
  亞太地區的使用者應使用專用網域 `https://bearlab.space` 而不是 `https://bearlab.ai`。您的 `sk-...` 密鑰在兩個網域上的功能相同。詳見 [服務可用性](/guides/availability)。
</Note>

## 兩種協定，一個密鑰

- **OpenAI 相容** — `POST <base_url>/v1/chat/completions`，使用 `Authorization: Bearer sk-...` 進行身份驗證。
- **Anthropic 相容** — `POST <base_url>/v1/messages`，傳送 `x-api-key: sk-...` 和 `anthropic-version: 2023-06-01`。

<Tabs>
  <Tab title="OpenAI 相容 (美國 / 歐洲)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-5.5",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="OpenAI 相容 (亞太地區)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-5.5",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="Anthropic 相容 (美國 / 歐洲)">

    curl https://bearlab.ai/v1/messages \
      -H "x-api-key: sk-..." \

      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-opus-4-7",
        "max_tokens": 256,

  </Tab>
