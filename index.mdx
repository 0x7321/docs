## 服務區域

BearLab AI面向**美國與歐盟**司法管轄區直接提供服務。

<Note>
**亞太區域**由區域合作伙伴提供服務，該區域的 API 請求須改用專用網域 `https://bearlab.in`，而非 `https://bearlab.ai`。該區域服務的相關政策由區域合作伙伴自行制定，亞太區域使用者請自行確保符合所在地的法律法規。
</Note>

| 區域 | API 域名 | OpenAI | Anthropic |
| --- | --- | --- | --- |
| 美國 / 歐盟 | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| 亞太區域 | `https://bearlab.in` | `https://bearlab.in/v1/chat/completions` | `https://bearlab.in/v1/messages` |

詳見 [服務區域](/guides/availability)。

BearLab AI（BearLab.ai）把數十家廠商的對話模型收到一個 API 下。申請一個 Key、配一個 `base_url`，就能調 OpenAI、Anthropic、Google、DeepSeek 的最新對話模型，不用各家分別註冊和充值。

介面同時相容 OpenAI 和 Anthropic 兩套協議，現有程式碼基本不用改。
