北极熊 AI（BearLabs.ai）把数十家厂商的对话模型收到一个 API 下。申请一个 Key、配一个 `base_url`，就能调 OpenAI、Anthropic、Google、DeepSeek 的最新对话模型，不用各家分别注册和充值。

接口同时兼容 OpenAI 和 Anthropic 两套协议，现有代码基本不用改。
