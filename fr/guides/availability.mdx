---
title: "Service Availability"
description: "Où BearLab sert directement, et comment la région Asie-Pacifique se connecte."
---

BearLab (bearlab.ai) fournit des services directement dans les juridictions des **États-Unis et de l'Union européenne**.

La région Asie-Pacifique est desservie par un **Partenaire régional**. Les requêtes API de cette région doivent utiliser le domaine dédié **`https://bearlab.in`** au lieu de `https://bearlab.ai`. Les politiques de service dans cette région sont définies par le Partenaire régional. Les utilisateurs de la région Asie-Pacifique sont responsables de la conformité aux lois locales applicables.

| Région | Domaine API |
| --- | --- |
| US / EU | `https://bearlab.ai` |
| Asie-Pacifique | `https://bearlab.in` |

base_url OpenAI = domaine + `/v1` (endpoint `/v1/chat/completions`) ; base_url Anthropic = domaine (endpoint `/v1/messages`). Voir [Chat Completions](/fr/guides/chat).
