[Codex CLI](https://github.com/openai/codex) est l'agent de codage officiel en ligne de commande d'OpenAI, connecté via un fournisseur personnalisé.

## Configuration

Modifiez `~/.codex/config.toml` et ajoutez un fournisseur pointant vers BearLab :

```toml
model = "gpt-4o"
model_provider = "bearlab"

# Third-party providers must disable response storage
disable_response_storage = true

[model_providers.bearlab]
name = "BearLab"
base_url = "https://bearlab.ai/v1"
env_key = "BEARLAB_API_KEY"
wire_api = "responses"
```

Ensuite exportez la clé et exécutez :

```bash
export BEARLAB_API_KEY="YOUR_API_KEY"
codex
```

<Warning>
  Utilisez l'adresse compatible OpenAI (`/v1`). Les fournisseurs tiers doivent définir `wire_api = "responses"` (le seul protocole pris en charge par Codex) et `disable_response_storage = true`, sinon certaines fonctionnalités échouent. `model` doit exister dans le [point de terminaison des modèles](/fr/guides/models).
</Warning>
