---
title: "Cursor"
description: "Utilisez les modèles conversationnels de BearLab dans Cursor."
---

[Cursor](https://cursor.com) est un éditeur de code IA populaire. Connectez-le via l'interface compatible OpenAI.

<Steps>
  <Step title="Ouvrir les paramètres du modèle">
    Ouvrez Settings, Models.
  </Step>
  <Step title="Remplacer l'URL de base">
    Dans la zone Clé API OpenAI, développez Override OpenAI Base URL et définissez :
    - **Base URL**: `https://bearlab.ai/v1`
    - **API Key**: votre clé `sk-...`
  </Step>
  <Step title="Ajoutez des modèles et vérifiez">
    Ajoutez les noms de modèle que vous voulez (par ex. `gpt-4o`, `claude-sonnet-4-6`), enregistrez, puis cliquez sur Verify.
  </Step>
</Steps>

<Note>
  Cursor utilise uniquement le protocole compatible OpenAI, donc l'URL de base se termine par `/v1`. Les noms de modèles doivent exister dans le [point de terminaison des modèles](/fr/guides/models). Si la vérification échoue, confirmez la clé avec le curl du [Démarrage rapide](/fr/guides/quickstart) d'abord.
</Note>
