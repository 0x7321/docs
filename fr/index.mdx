BearLab AI (bearlab.ai) réunit les modèles conversationnels de dizaines de fournisseurs sous une seule API. Créez une clé, définissez une `base_url`, et appelez les derniers modèles d'OpenAI, Anthropic, Google, DeepSeek et autres sans vous inscrire et recharger séparément chez chaque fournisseur.

L'interface est compatible avec les protocoles OpenAI et Anthropic, ce qui signifie que le code existant fonctionne généralement avec une simple modification `base_url` sur une ligne.

## URLs de base par région

Choisissez l'URL de base pour votre région et utilisez la même clé API sur les deux.

| Région | Domaine API | Compatible OpenAI | Compatible Anthropic |
| --- | --- | --- | --- |
| États-Unis / Europe | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| Asie-Pacifique | `https://bearlab.space` | `https://bearlab.space/v1/chat/completions` | `https://bearlab.space/v1/messages` |

<Note>
  Les utilisateurs d'Asie-Pacifique doivent utiliser le domaine dédié `https://bearlab.space` au lieu de `https://bearlab.ai`. Votre clé `sk-...` fonctionne de la même manière sur les deux. Voir [Disponibilité du service](/fr/guides/availability).
</Note>

## Deux protocoles, une clé

- **Compatible OpenAI** — `POST <base_url>/v1/chat/completions`, authentifiez-vous avec `Authorization: Bearer sk-...`.
- **Compatible Anthropic** — `POST <base_url>/v1/messages`, envoyez `x-api-key: sk-...` et `anthropic-version: 2023-06-01`.
