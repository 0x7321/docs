チャット補完はコアエンドポイントで、3つの互換インターフェース（OpenAI、Anthropic、Gemini）を通じて提供され、1つのキーを共有します。

<Note>
  BearLab AI は2つのリージョンをサービスしています。米国/EU ユーザーは `https://bearlab.ai` を呼び出します。アジア太平洋ユーザーは専用ドメイン `https://bearlab.space` を呼び出します。2つのリージョンは同じキーとモデルカタログを共有します。[サービス可用性](/ja/guides/availability) を参照。
</Note>

## エンドポイント

| プロトコル | メソッドとパス（米国/EU） | メソッドとパス（アジア太平洋） |
| --- | --- | --- |
| OpenAI互換 | `POST https://bearlab.ai/v1/chat/completions` | `POST https://bearlab.space/v1/chat/completions` |
| Anthropic互換 | `POST https://bearlab.ai/v1/messages` | `POST https://bearlab.space/v1/messages` |
| Gemini ネイティブ | `base_url=https://bearlab.ai`, `api_version=v1beta`（Google genai SDK） | `base_url=https://bearlab.space`, `api_version=v1beta` |

すべてのリクエストを `Authorization: Bearer` ヘッダー内の API キー（`sk-...`）で認証します。Anthropic プロトコルは `x-api-key` ヘッダーも受け入れます。

## 一般的なパラメータ

| パラメータ | 必須 | 説明 |
| --- | --- | --- |
| `model` | はい | モデル ID。[モデルと価格 API](/ja/guides/models) を参照。 |
| `messages` | はい | 順序付きの会話メッセージの配列。 |
| `max_tokens` | OpenAI：オプション · Anthropic：**必須** | 最大出力トークン数。 |
| `temperature` | いいえ | サンプリング温度。OpenAI の範囲 `0`-`2`、Anthropic の範囲 `0`-`1`。 |
| `stream` | いいえ | `true` の場合、SSE 経由で応答をストリーミングします。 |
| `tools` | いいえ | ツール呼び出しのための関数/ツール定義。 |

<Warning>
  プロトコルの違い：OpenAI プロトコルはシステムプロンプトを `messages` の `role: system` として配置し、`max_tokens` はオプションです。Anthropic プロトコルはトップレベルの `system` フィールドを使用し、`max_tokens` が**必須**です。
</Warning>

## ストリーミング

`stream: true` を設定して、Server-Sent Events 経由で応答を受信します。

- **OpenAI プロトコル** — 増分 `delta` チャンク、最終 `data: [DONE]` 行で終了。
- **Anthropic プロトコル** — 型付き SSE イベント（`message_start`、`content_block_delta`、`message_stop` など）；`[DONE]` センチネルなし。

<CodeGroup>
```python 米国/EU
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.ai/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```

```python アジア太平洋
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.space/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```
</CodeGroup>

## ツール呼び出し

OpenAI と Anthropic プロトコルの両方がツール呼び出しをサポートしています。

```python
from openai import OpenAI

client = OpenAI(base_url="https://bearlab.ai/v1", api_key="sk-...")

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the current weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"],
        },
    },
}]

resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "What's the weather in London?"}],
    tools=tools,
)
print(resp.choices[0].message)
```

<Note>
  アジア太平洋ユーザー：代わりに `base_url="https://bearlab.space/v1"` を設定してください。
</Note>

## Gemini ネイティブプロトコル

Gemini モデルは、OpenAI 形式に変換することなく、ネイティブ Google AI Studio 形式で呼び出すことができます。

<CodeGroup>
```python 米国/EU
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.ai"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```

```python アジア太平洋
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.space"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```
</CodeGroup>

<Warning>
  1. `base_url` をリージョンルート（`https://bearlab.ai` または `https://bearlab.space`）に設定します。`/v1beta` を追加しないでください。SDK が追加します。
  2. `api_version` を `v1beta` に設定します。
  3. インストール：Python `pip install google-genai`、Node.js `npm install @google/genai`。
</Warning>

<Tip>
  長い繰り返しプレフィックス（固定システムプロンプト、長いドキュメント）は、[プロンプトキャッシング](/ja/guides/prompt-caching) を使用して、繰り返し部分のコストを削減できます。
</Tip>
