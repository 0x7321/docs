---
title: "サービス提供地域"
description: "BearLabが直接サービスする場所、およびアジア太平洋地域がどのように接続するか。"
---

BearLab (bearlab.ai) は、**米国およびヨーロッパ連合**の管轄区域にサービスを直接提供しています。

アジア太平洋地域は**地域パートナー**によってサービスされます。この地域からのAPIリクエストは、`https://bearlab.ai` の代わりに専用ドメイン **`https://bearlab.in`** を使用する必要があります。このリージョンのサービスのポリシーは地域パートナーによって設定されます。アジア太平洋地域のユーザーは、適用される現地法令への準拠を責任を負います。

| 地域 | API ドメイン |
| --- | --- |
| 米国 / EU | `https://bearlab.ai` |
| アジア太平洋 | `https://bearlab.in` |

OpenAI の base_url = ドメイン + `/v1`(エンドポイント `/v1/chat/completions`)、Anthropic の base_url = ドメイン(エンドポイント `/v1/messages`)。[チャット完了](/ja/guides/chat) を参照。
