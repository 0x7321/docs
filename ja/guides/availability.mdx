title: "サービス提供地域"
description: "BearLabが直接サービスする場所、およびアジア太平洋地域がどのように接続するか。"
---

BearLab (bearlab.ai) は、**米国およびヨーロッパ連合**の管轄区域にサービスを直接提供しています。

### アジア太平洋

アジア太平洋地域は地域パートナーによってサービスされます。

この地域からの API リクエストは、`https://bearlab.ai` の代わりに専用ドメイン `https://bearlab.in` を使用する必要があります。

このリージョンのサービスのポリシーは地域パートナーによって設定され、ユーザーは適用される現地法令への準拠に責任を負います。

| 地域 | API ドメイン | OpenAI | Anthropic |
| --- | --- | --- | --- |
| 米国 / EU | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| アジア太平洋 | `https://bearlab.in` | `https://bearlab.in/v1/chat/completions` | `https://bearlab.in/v1/messages` |
