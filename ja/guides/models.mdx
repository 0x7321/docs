---

`/v1/models/pricing` は、現在利用可能なすべての会話型モデルとその公表価格を返す**公開の認証不要**な集約エンドポイントです。レスポンスは[OpenRouterプロバイダー仕様](https://openrouter.ai/docs/guides/community/for-providers)に従うため、OpenRouter、サードパーティの価格集約ツール、またはカスタムクライアントに取り込めます。

<Note>
  アジア太平洋ユーザー: `https://bearlab.ai` の代わりに専用ドメイン `https://bearlab.in` を使用してください。[サービス提供地域](/ja/guides/availability)を参照してください。
</Note>

<Note>
  これは「公表価格」エントリーポイントであり、ユーザー身元を含まないものです。価格は匿名コンテキストの`default`グループを表しています。認証されたユーザーが実際に支払う価格(グループ、階層割引など)を確認するには、コンソールの価格ページを使用してください。
</Note>

## 使用する場合

| ユースケース | 説明 |
| --- | --- |
| 価格集約 | OpenRouterおよび他の集約ツールがカタログと公表価格をスクレイピングします |
| クライアント表示 | アプリ、ボット、SDKがモデルリストをレンダリングします |
| モニタリング | 時間経過に伴うカタログまたは価格の変更を追跡します |

## エンドポイント

| 項目 | 値 |
| --- | --- |
| **URL** | `https://bearlab.ai/v1/models/pricing` |
| **メソッド** | `GET` |
| **認証** | なし(公開) |
| **レート制限** | IP単位の厳格な臨界レート制限 |
| **キャッシング** | 60秒のインプロセスキャッシュ; `Cache-Control: public, max-age=60` |

### リクエスト

```bash
curl https://bearlab.ai/v1/models/pricing
```

## レスポンス構造

最上位は常に`data`配列で囲まれています。

```json
{
  "data": [
    {
      "id": "gpt-4o",
      "name": "GPT-4o",
      "created": 1715558400,
      "input_modalities": ["text"],
      "output_modalities": ["text"],
      "context_length": 128000,
      "max_output_length": 16384,
      "pricing": {
        "prompt": "0.0000025",
        "completion": "0.00001",
        "request": "0",
        "input_cache_read": "0.00000125"
      },
      "supported_features": ["tools", "json_mode", "structured_outputs", "logprobs"]
    }
  ]
}
```

### モデルフィールド

| フィールド | タイプ | 説明 |
| --- | --- | --- |
| `id` | `string` | ユニークなモデル識別子; `/v1/chat/completions`を呼び出すときに使用します |
| `name` | `string` | 表示名; `id`へフォールバック |
| `created` | `int64` | リリース時間(Unixセコンド) |
| `input_modalities` / `output_modalities` | `string[]` | 少なくとも`text` |
| `context_length` | `int` | トークン内の最大コンテキストウィンドウ |
| `max_output_length` | `int` | レスポンスごとの最大出力トークン |
| `pricing` | `Pricing` | 価格オブジェクト。以下を参照してください |
| `pricing_tiers` | `Pricing[]` | 階層化された価格(最大1つ、コンテキスト長の閾値による) |
| `supported_features` | `string[]` | 機能フラグ |

### 価格フィールド

すべての価格は**USDの文字列**です。トークンクラスフィールドはトークンごとに価格が設定されます。

| フィールド | 単位 | 説明 |
| --- | --- | --- |
| `prompt` | USD / 入力トークン | 入力単価 |
| `completion` | USD / 出力トークン | 出力単価 |
| `request` | USD / リクエスト | リクエストごとの固定料金(チャットモデルでは通常`"0"`) |
| `internal_reasoning` | USD / 推論トークン | 推論トークンの価格 |
| `input_cache_read` | USD / トークン | キャッシュヒット読み取り価格(通常はより低い) |
| `input_cache_write` | USD / トークン | キャッシュ書き込み価格 |
| `web_search` | USD / クエリ | 組み込みWebサーチツール料金 |

<Note>
  `"0"`は「無料」または「このモデルに適用不可」を意味する場合があります。価格だけに頼らず、`supported_features`と組み合わせて機能を推測してください。
</Note>

### `supported_features`

| 値 | 意味 |
| --- | --- |
| `tools` | 関数呼び出し / ツール使用 |
| `reasoning` | 推論トークンを返す |
| `json_mode` | JSON モード出力 |
| `structured_outputs` | スキーマ制約付きの構造化出力 |
| `logprobs` | トークンログ確率 |
| `web_search` | 組み込みWebサーチツール |

## キャッシングと更新頻度

| アスペクト | 動作 |
| --- | --- |
| サーバーキャッシュ | 60秒のインプロセス; 管理者更新は1分以内に反映されます |
| クライアントキャッシュ | `Cache-Control: public, max-age=60` |
| 推奨ポーリング | 最多でも5分ごと; 積極的なポーリングはレート制限されます |

## 他のエンドポイントとの比較

| エンドポイント | 目的 | 認証 |
| --- | --- | --- |
| `/v1/models/pricing` | OpenRouter形式の公開価格 | 公開 |
| `/v1/models` | OpenAI SDK互換モデルIDリスト | トークン必須 |
