このページは、お使いの API キーで呼び出せるモデルを、ファミリー別に分類して一覧にしています。チャットリクエストの `model` フィールドに、モデルの確切な `id` を渡してください。

<Note>
  アジア太平洋地域のユーザーの皆様：`https://bearlab.ai` の代わりに、専用ドメイン `https://bearlab.space` をお使いください。[サービス可用性](/ja/guides/availability) をご覧ください。
</Note>

## このカタログの読み方

ご覧いただけるカタログは、**ご利用のトークングループ** に基づいています。各 API キーは 1 つのモデルグループに属しており、モデルはご利用のグループが呼び出しを許可されている場合のみ、リストに表示されます。以下のモデルは現在の本番ラインアップです。実際にキーで表示されるサブセットはグループによります。

| 概念 | 説明 |
| --- | --- |
| **モデル `id`** | `model` フィールドで送信する正確な文字列です。大文字と小文字が区別されます。そのままコピーしてください。 |
| **モデルグループ** | ご利用のキーが属するアクセス層です。呼び出せるモデルと価格を決定します。 |
| **エンドポイントタイプ** | モデルが受け入れるプロトコル：OpenAI 互換、Anthropic、または Gemini です。 |

## ライブリストを取得する

`/v1/models` は、お使いのキーが呼び出せるモデルを OpenAI SDK 形式で返します。API キーが必要です。レスポンス形式はご利用のリクエストヘッダーで示唆される形式に従います（デフォルトは OpenAI。`x-api-key` + `anthropic-version` の場合は Anthropic。`x-goog-api-key` の場合は Gemini）。

<Tabs>
  <Tab title="米国 / EU (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
  <Tab title="アジア太平洋地域 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
</Tabs>

レスポンス（省略）：

```json
{
  "object": "list",
  "data": [
    {
      "id": "claude-opus-4-8",
      "object": "model",
      "created": 1626777600,
      "owned_by": "openai",
      "supported_endpoint_types": ["openai"]
    }
  ]
}
```

| フィールド | 説明 |
| --- | --- |
| `id` | モデル識別子。チャットリクエストの `model` フィールドで渡してください |
| `object` | 常に `model` |
| `created` | リリース時間（Unix 秒） |
| `owned_by` | 名義上の所有者文字列 |
| `supported_endpoint_types` | このモデルを受け入れるエンドポイント：`openai`、`anthropic`、および/または `gemini` |

## モデルを呼び出す

`model` フィールドに正確な `id` を送信してください。両方のサンプルドメインを表示しています。地域に合わせて選択してください。

<Tabs>
  <Tab title="米国 / EU (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="アジア太平洋地域 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
</Tabs>

<Tip>
  価格と計費ルールについては、[価格と計費](/ja/guides/pricing) を参照してください。グループが割引にどのようにマップするかについては、[ユーザーレベル](/ja/guides/user-levels) を参照してください。
</Tip>
