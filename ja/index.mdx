BearLab AI (bearlab.ai) は、数十のプロバイダーの会話型モデルを1つのAPIで統合します。1つのキーを作成し、1つの`base_url`を設定すれば、OpenAI、Anthropic、Google、DeepSeekおよび他のプロバイダーの最新モデルを、各ベンダーで個別に登録したり入金したりすることなく呼び出せます。

このインターフェースはOpenAIおよびAnthropicのプロトコルの両方に対応しているため、既存のコードは通常、1行の`base_url`変更で動作します。

## 地域別 base URL

ご利用の地域に応じてbase URLを選択し、同じAPIキーを両方で使用できます。

| 地域 | API ドメイン | OpenAI 互換 | Anthropic 互換 |
| --- | --- | --- | --- |
| 米国 / ヨーロッパ | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| アジア太平洋 | `https://bearlab.space` | `https://bearlab.space/v1/chat/completions` | `https://bearlab.space/v1/messages` |

<Note>
  アジア太平洋のユーザーは、`https://bearlab.ai` の代わりに専用ドメイン `https://bearlab.space` を使用してください。`sk-...` キーは両方のドメインで同じように動作します。[サービス可用性](/ja/guides/availability)を参照してください。
</Note>

## 2つのプロトコル、1つのキー

- **OpenAI 互換** — `POST <base_url>/v1/chat/completions`、`Authorization: Bearer sk-...` で認証します。
- **Anthropic 互換** — `POST <base_url>/v1/messages`、`x-api-key: sk-...` と `anthropic-version: 2023-06-01` を送信します。

<Tabs>
  <Tab title="OpenAI 互換 (米国 / ヨーロッパ)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-5.5",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="OpenAI 互換 (アジア太平洋)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-5.5",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="Anthropic 互換 (米国 / ヨーロッパ)">
    ```bash
    curl https://bearlab.ai/v1/messages \
      -H "x-api-key: sk-..." \
      -H "anthropic-version: 2023-06-01" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-opus-4-7",
        "max_tokens": 256,
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
