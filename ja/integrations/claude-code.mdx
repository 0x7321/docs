[Claude Code](https://docs.anthropic.com/ja/docs/claude-code/overview) はAnthropicの公式AIコーディングCLIで、Anthropicプロトコルをネイティブに使用して接続します。

## サポートモデル

| モデル | ID | 用途 |
| --- | --- | --- |
| Claude Opus 4.8 | `claude-opus-4-8` | 最高推論、複雑なアーキテクチャ |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | バランス性能、日常開発向け |
| Claude Haiku 4.5 | `claude-haiku-4-5` | 最速、軽量補完・クイック応答 |

## 設定

環境変数をプラットフォームエンドポイントに向けます:

```bash
export ANTHROPIC_BASE_URL="https://bearlab.ai"
export ANTHROPIC_API_KEY="YOUR_API_KEY"

claude
```

`ANTHROPIC_API_KEY` と `ANTHROPIC_AUTH_TOKEN` はどちらも認識され、完全に同等です。どちらでも使用できます。

## モデルを指定する（オプション）

```bash
export ANTHROPIC_MODEL="claude-sonnet-4-6"           # メインモデルをオーバーライド
export ANTHROPIC_SMALL_FAST_MODEL="claude-haiku-4-5" # 内部高速サブタスク用モデルをオーバーライド
```

設定しない場合、Claude Code は自身のデフォルトモデル選択ロジックを使用します。

## 永続化

```bash
echo 'export ANTHROPIC_BASE_URL="https://bearlab.ai"' >> ~/.zshrc
echo 'export ANTHROPIC_API_KEY="YOUR_API_KEY"' >> ~/.zshrc
```

<Note>
  `ANTHROPIC_BASE_URL` にはプラットフォームのルートドメインを設定します（`/v1` は不要）。モデルIDは[modelsエンドポイント](/ja/guides/models)に存在するものを使用してください。
</Note>
