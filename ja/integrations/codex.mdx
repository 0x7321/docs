[Codex CLI](https://github.com/openai/codex) はOpenAIの公式コマンドラインコーディングエージェントで、カスタムプロバイダー経由で接続されます。

## 推奨モデル

| モデル | ID | 用途 |
| --- | --- | --- |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | コーディング向け、速度と能力のバランス |
| o4-mini | `o4-mini` | 高速推論、デバッグと説明に最適 |
| GPT-4o | `gpt-4o` | 汎用コーディングタスク |

## 設定

`~/.codex/config.toml`を編集し、BearLabを指すプロバイダーを追加します:

```toml
model = "claude-sonnet-4-6"
model_provider = "bearlab"

# サードパーティプロバイダーはレスポンスストレージを無効にする必要があります
disable_response_storage = true

[model_providers.bearlab]
name = "BearLab AI"
base_url = "https://bearlab.ai/v1"
env_key = "BEARLAB_API_KEY"
wire_api = "responses"
```

キーをエクスポートして実行します:

```bash
export BEARLAB_API_KEY="YOUR_API_KEY"
codex
```

<Note>
  `/v1` で終わるOpenAI互換アドレスを使用してください。サードパーティプロバイダーは `wire_api = "responses"` と `disable_response_storage = true` を設定する必要があります。`model` は[modelsエンドポイント](/ja/guides/models)に存在するものを使用してください。
</Note>
