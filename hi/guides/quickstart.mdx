एक model को कॉल करने के लिए तीन steps।

<Note>
  Asia-Pacific users: `https://bearlab.ai` के बजाय समर्पित डोमेन `https://bearlab.in` का उपयोग करें। [Service Availability](/hi/guides/availability) देखें।
</Note>

<Steps>
  <Step title="Sign up और top up करें">
    [console](https://bearlab.ai) पर register करें और Wallet page पर funds add करें। नए users एक छोटे top-up के साथ test शुरू कर सकते हैं।
  </Step>
  <Step title="एक API key बनाएं">
    Dashboard, API Tokens पर जाएं, Add Token पर click करें, और `sk-...` key को copy करें। Key पूरी तरह से केवल एक बार दिखाया जाता है।
  </Step>
  <Step title="अपना पहला request बनाएं">
    `YOUR_API_KEY` को अपनी key से replace करें।
  </Step>
</Steps>

## उदाहरण

<Tabs>
  <Tab title="OpenAI-compatible">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-4o",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="Anthropic-compatible">
    ```bash
    curl https://bearlab.ai/v1/messages \
      -H "x-api-key: YOUR_API_KEY" \
      -H "anthropic-version: 2023-06-01" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "max_tokens": 256,
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="Python (OpenAI SDK)">
    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://bearlab.ai/v1",
    )

    resp = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "Introduce yourself in one sentence"}],
    )
    print(resp.choices[0].message.content)
    ```
  </Tab>
</Tabs>
