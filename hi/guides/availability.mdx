title: "Service Availability"
description: "जहां BearLab सीधे serves करता है, और Asia-Pacific region कैसे connect करता है।"
---

BearLab (bearlab.ai) **United States और European Union** jurisdictions में सीधे services provide करता है।

Asia-Pacific region एक **Regional Partner** द्वारा serve किया जाता है। इस region से API requests को `https://bearlab.ai` के बजाय समर्पित डोमेन **`https://bearlab.in`** का उपयोग करना चाहिए। इस region में service के लिए policies Regional Partner द्वारा set किए जाते हैं। Asia-Pacific region में users applicable local laws के साथ compliance के लिए responsible हैं।

| Region | API domain |
| --- | --- |
| US / EU | `https://bearlab.ai` |
| Asia-Pacific | `https://bearlab.in` |

OpenAI base_url = domain + `/v1` (endpoint `/v1/chat/completions`); Anthropic base_url = domain (endpoint `/v1/messages`). [Chat Completions](/hi/guides/chat) देखें।
