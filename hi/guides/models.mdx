यह पृष्ठ आपकी API कुंजी द्वारा कॉल किए जा सकने वाले मॉडलों को परिवार द्वारा समूहीकृत करके सूचीबद्ध करता है। चैट अनुरोध के `model` फ़ील्ड में मॉडल का सटीक `id` पास करें।

<Note>
  एशिया-प्रशांत उपयोगकर्ता: `https://bearlab.ai` के बजाय समर्पित डोमेन `https://bearlab.space` का उपयोग करें। [सेवा उपलब्धता](/hi/guides/availability) देखें।
</Note>

## इस कैटलॉग को कैसे पढ़ें

जो कैटलॉग आप देखते हैं वह **आपके टोकन समूह** के लिए बंधा हुआ है। प्रत्येक API कुंजी एक मॉडल समूह के अंतर्गत आती है, और एक मॉडल केवल तभी आपकी सूची में दिखाई देता है जब आपका समूह इसे कॉल करने के लिए अनुमति दी जाती है। नीचे दिए गए मॉडल वर्तमान उत्पादन लाइनअप हैं; आपकी कुंजी के लिए दिखाया गया सटीक सबसेट आपके समूह पर निर्भर करता है।

| अवधारणा | अर्थ |
| --- | --- |
| **मॉडल `id`** | वह सटीक स्ट्रिंग जो आप `model` में भेजते हैं। केस-संवेदनशील, इसे शब्दशः कॉपी करें। |
| **मॉडल समूह** | एक्सेस टियर जिसके तहत आपकी कुंजी आती है। निर्धारित करता है कि आप कौन से मॉडल कॉल कर सकते हैं और किस कीमत पर। |
| **एंडपॉइंट प्रकार** | कौन सी प्रोटोकॉल एक मॉडल स्वीकार करता है: OpenAI-संगत, Anthropic, या Gemini। |

## लाइव सूची प्राप्त करें

`/v1/models` उन मॉडलों को लौटाता है जिन्हें आपकी कुंजी कॉल कर सकती है, OpenAI SDK प्रारूप में। इसके लिए आपकी API कुंजी की आवश्यकता है। प्रतिक्रिया प्रारूप आपके अनुरोध शीर्षलेख द्वारा निहित प्रारूप का अनुसरण करता है (डिफ़ॉल्ट रूप से OpenAI; `x-api-key` + `anthropic-version` के साथ Anthropic; `x-goog-api-key` के साथ Gemini)।

<Tabs>
  <Tab title="यूएस / यूई (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
  <Tab title="एशिया-प्रशांत (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
</Tabs>

प्रतिक्रिया (छंटनी की गई):

```json
{
  "object": "list",
  "data": [
    {
      "id": "claude-opus-4-8",
      "object": "model",
      "created": 1626777600,
      "owned_by": "openai",
      "supported_endpoint_types": ["openai"]
    }
  ]
}
```

| फ़ील्ड | विवरण |
| --- | --- |
| `id` | मॉडल पहचानकर्ता; चैट अनुरोध के `model` फ़ील्ड में यह पास करें |
| `object` | हमेशा `model` |
| `created` | रिलीज़ समय (Unix सेकंड) |
| `owned_by` | नाममात्र मालिक स्ट्रिंग |
| `supported_endpoint_types` | कौन से एंडपॉइंट इस मॉडल को स्वीकार करते हैं: `openai`, `anthropic`, और/या `gemini` |

## एक मॉडल को कॉल करें

`model` फ़ील्ड में सटीक `id` भेजें। दोनों उदाहरण डोमेन दिखाए गए हैं; अपने क्षेत्र के लिए सही चुनें।

<Tabs>
  <Tab title="यूएस / यूई (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="एशिया-प्रशांत (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
</Tabs>
