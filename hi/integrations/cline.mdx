---
title: "Cline / Roo Code"
description: "Cline या Roo Code VS Code extensions में BearLab AI का उपयोग करें।"
---

[Cline](https://github.com/cline/cline) और इसके enhanced fork [Roo Code](https://github.com/RooVetGit/Roo-Code) VS Code AI coding extensions हैं। उन्हें OpenAI-compatible interface के माध्यम से BearLab AI से जोड़ें।

<Steps>
  <Step title="Provider चुनें">
    Extension settings में, API Provider को **OpenAI Compatible** पर set करें।
  </Step>
  <Step title="Connection details भरें">
    - **Base URL** : आपके region के लिए endpoint का उपयोग करें

      <Tabs>
        <Tab title="US / EU">
          ```
          https://bearlab.ai/v1
          ```
        </Tab>
        <Tab title="Asia-Pacific">
          ```
          https://bearlab.space/v1
          ```
        </Tab>
      </Tabs>

    - **API Key** : आपकी `sk-...` key
    - **Model ID** : [models endpoint](/hi/guides/models) से exact model id, जैसे `claude-sonnet-4-6`, `gpt-5.4` या `deepseek-v4-pro`
  </Step>
</Steps>

<Note>
  **OpenAI Compatible** चुनें, Anthropic नहीं, और Base URL पर `/v1` suffix रखें। Cline आपकी key के साथ `Authorization: Bearer sk-...` header में `POST /v1/chat/completions` को call करता है।
</Note>

<Warning>
  **Model ID** को [models endpoint](/hi/guides/models) से return किए गए id से exactly match करना चाहिए। `gpt-4o` जैसे नाम जो उस list में नहीं हैं, valid model ids नहीं हैं और requests reject हो जाएंगे।
</Warning>
