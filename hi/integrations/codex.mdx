[Codex CLI](https://github.com/openai/codex) OpenAI का official command-line coding agent है, एक custom provider के माध्यम से connected।

## Configuration

`~/.codex/config.toml` को edit करें और BearLab को point करने वाला एक provider add करें:

```toml
model = "gpt-4o"
model_provider = "bearlab"

# Third-party providers को response storage को disable करना चाहिए
disable_response_storage = true

[model_providers.bearlab]
name = "BearLab"
base_url = "https://bearlab.ai/v1"
env_key = "BEARLAB_API_KEY"
```

फिर key को export करें और run करें:

```bash
export BEARLAB_API_KEY="YOUR_API_KEY"
codex
```

<Warning>
  OpenAI-compatible address (`/v1`) का उपयोग करें। Third-party providers के लिए `disable_response_storage = true` set करें, अन्यथा कुछ features error होते हैं। `model` [models endpoint](/hi/guides/models) में exist करने चाहिए।
</Warning>
