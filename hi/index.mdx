## सेवा उपलब्धता

BearLab **संयुक्त राज्य अमेरिका और यूरोपीय संघ** jurisdictions को सीधे सेवा प्रदान करता है।

<Tip>
**एशिया-प्रशांत** क्षेत्र एक Regional Partner द्वारा सेवा प्रदान किया जाता है, और उस क्षेत्र से API requests के लिए `https://bearlab.ai` के बजाय समर्पित डोमेन `https://bearlab.in` का उपयोग करना चाहिए।
</Tip>

| Region | API domain |
| --- | --- |
| US / EU | `https://bearlab.ai` |
| Asia-Pacific | `https://bearlab.in` |

OpenAI base_url = domain + `/v1` (endpoint `/v1/chat/completions`); Anthropic base_url = domain (endpoint `/v1/messages`). [Chat Completions](/hi/guides/chat) देखें।

[Service Availability](/hi/guides/availability) देखें।

BearLab (BearLab.ai) दर्जनों providers के conversational models को एक API के तहत लाता है। एक key बनाएं, एक `base_url` सेट करें, और OpenAI, Anthropic, Google, और DeepSeek के नवीनतम models को बिना प्रत्येक vendor के साथ अलग से रजिस्टर और top-up किए कॉल करें।

यह interface OpenAI और Anthropic दोनों protocols के साथ compatible है, इसलिए मौजूदा code आमतौर पर एक-पंक्ति `base_url` परिवर्तन के साथ काम करता है।
