BearLab AI (bearlab.ai) दर्जनों providers के conversational models को एक API के तहत लाता है। एक key बनाएं, एक `base_url` सेट करें, और OpenAI, Anthropic, Google, DeepSeek और अन्य के नवीनतम models को बिना प्रत्येक vendor के साथ अलग से रजिस्टर और top-up किए कॉल करें।

यह interface OpenAI और Anthropic दोनों protocols के साथ compatible है, इसलिए मौजूदा code आमतौर पर एक-पंक्ति `base_url` परिवर्तन के साथ काम करता है।

## क्षेत्रीय base URLs

अपने region के लिए base URL चुनें और दोनों में एक ही API key का उपयोग करें।

| Region | API Domain | OpenAI-compatible | Anthropic-compatible |
| --- | --- | --- | --- |
| US / Europe | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| Asia-Pacific | `https://bearlab.space` | `https://bearlab.space/v1/chat/completions` | `https://bearlab.space/v1/messages` |

<Note>
  Asia-Pacific users को `https://bearlab.ai` के बजाय समर्पित domain `https://bearlab.space` का उपयोग करना चाहिए। आपकी `sk-...` key दोनों domains पर समान रूप से काम करती है। [Service Availability](/hi/guides/availability) देखें।
</Note>

## दो protocols, एक key

- **OpenAI-compatible** — `POST <base_url>/v1/chat/completions`, `Authorization: Bearer sk-...` से authenticate करें।
- **Anthropic-compatible** — `POST <base_url>/v1/messages`, `x-api-key: sk-...` और `anthropic-version: 2023-06-01` भेजें।
