本頁列出您的 API 金鑰可以呼叫的模型，按系列分組。在聊天請求的 `model` 欄位中傳入模型的確切 `id`。

<Note>
  亞太地區使用者：請改用專用網域 `https://bearlab.space` 而非 `https://bearlab.ai`。詳見 [服務可用性](/guides/availability)。
</Note>

## 如何閱讀本目錄

您看到的目錄受限於 **您的權杖群組**。每個 API 金鑰屬於一個模型群組，模型只有在您的群組被允許呼叫時才會出現在您的清單中。下方的模型是當前的正式生產陣容；您的金鑰實際看到的子集取決於您的群組。

| 概念 | 意義 |
| --- | --- |
| **模型 `id`** | 您在 `model` 欄位中傳送的確切字串。區分大小寫，請逐字複製。 |
| **模型群組** | 您的金鑰所屬的存取層級。決定您可以呼叫哪些模型以及以什麼價格。 |
| **端點類型** | 模型接受的協議：OpenAI 相容、Anthropic 或 Gemini。 |

## 取得即時清單

`/v1/models` 會傳回您的金鑰可以呼叫的模型，採用 OpenAI SDK 格式。它需要您的 API 金鑰。回應格式遵循您的請求標頭所暗示的格式（預設為 OpenAI；使用 `x-api-key` + `anthropic-version` 時為 Anthropic；使用 `x-goog-api-key` 時為 Gemini）。

<Tabs>
  <Tab title="美國 / 歐盟 (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
  <Tab title="亞太地區 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
</Tabs>

回應（截斷）：

```json
{
  "object": "list",
  "data": [
    {
      "id": "claude-opus-4-8",
      "object": "model",
      "created": 1626777600,
      "owned_by": "openai",
      "supported_endpoint_types": ["openai"]
    }
  ]
}
```

| 欄位 | 描述 |
| --- | --- |
| `id` | 模型識別碼；在聊天請求的 `model` 欄位中傳入此值 |
| `object` | 始終為 `model` |
| `created` | 發佈時間（Unix 秒） |
| `owned_by` | 名義所有者字串 |
| `supported_endpoint_types` | 接受此模型的端點：`openai`、`anthropic` 和/或 `gemini` |

## 呼叫一個模型

在 `model` 欄位中傳送確切的 `id`。兩個範例網域均已顯示；請選擇適合您地區的那個。

<Tabs>
  <Tab title="美國 / 歐盟 (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="亞太地區 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
</Tabs>
