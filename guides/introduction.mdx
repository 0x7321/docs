> BearLab AI（BearLab.ai）把數十家主流廠商的對話模型收到一個 API 下，OpenAI、Anthropic、Google、DeepSeek 都能用同一個 Key 調。

## BearLab AI 是什麼

它是一個對話模型聚合平臺。你不用再去 OpenAI、Anthropic、Google 各自注冊、各自充值、各自管一套金鑰，在這裡申請一個 Key，配一個 `base_url`，就能調它們的最新對話模型。

介面同時相容 OpenAI 和 Anthropic 兩套協議，現有程式碼基本不用改，換個地址和金鑰就能接進來。

我們只做對話模型這一件事，所以介面簡單、計費清楚、跑得穩。

## 主要能力

**一個 Key 用所有模型**
不同廠商、不同模型，統一用一套介面呼叫。一個賬戶管許可權，一份賬單看花銷，一處看用量。

**OpenAI / Anthropic 雙協議**
兩套協議都原生支援，Cursor、Claude Code、Cline、Codex 這些工具直接填地址就能用，按需要切換，不用關心背後是哪家的模型。

**穩定可用**
熱門模型接多家供應商，某一家出問題自動切到別家，請求不中斷。

**按量付費**
用多少算多少，沒有月費和起充門檻，價格隨上游浮動，餘額實時可查。

**用量可查**
每次呼叫的請求、響應、Token 消耗都有記錄，方便排查問題和算成本。

## 只做對話模型

BearLab AI 不提供影象、語音、影片、音樂、嵌入（Embedding）這些模型，專注把對話模型做穩做透。需要那些能力請用對應的專門平臺。
