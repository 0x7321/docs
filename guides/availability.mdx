## 服務區域

BearLab AI（bearlab.ai）面向**美國與歐盟**司法管轄區直接提供服務。

亞太區域由**區域合作伙伴**提供服務：該區域的 API 請求須使用專用網域 **`https://bearlab.in`**，而非 `https://bearlab.ai`。該區域服務的相關政策由區域合作伙伴自行制定。亞太區域使用者請自行確保符合所在地的法律法規。

| 區域 | API 域名 | OpenAI | Anthropic |
| --- | --- | --- | --- |
| 美國 / 歐盟 | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| 亞太區域 | `https://bearlab.in` | `https://bearlab.in/v1/chat/completions` | `https://bearlab.in/v1/messages` |
