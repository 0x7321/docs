對話補全是核心端點，透過三種相容介面（OpenAI、Anthropic、Gemini）提供，共享一個密鑰。

<Note>
  BearLab AI 服務兩個地區。美國/歐洲使用者呼叫 `https://bearlab.ai`；亞太地區使用者呼叫專用網域 `https://bearlab.space`。兩個地區共享同一密鑰和模型目錄。見 [服務可用性](/guides/availability)。
</Note>

## 端點

| 協議 | 方法與路徑（美國/歐洲） | 方法與路徑（亞太地區） |
| --- | --- | --- |
| OpenAI 相容 | `POST https://bearlab.ai/v1/chat/completions` | `POST https://bearlab.space/v1/chat/completions` |
| Anthropic 相容 | `POST https://bearlab.ai/v1/messages` | `POST https://bearlab.space/v1/messages` |
| Gemini 原生 | `base_url=https://bearlab.ai`, `api_version=v1beta`（Google genai SDK） | `base_url=https://bearlab.space`, `api_version=v1beta` |

使用 API 密鑰（`sk-...`）在 `Authorization: Bearer` 標頭中驗證每個請求。Anthropic 協議另外接受 `x-api-key` 標頭。

## 常見參數

| 參數 | 必需 | 說明 |
| --- | --- | --- |
| `model` | 是 | 模型 ID。見 [模型與定價 API](/guides/models)。 |
| `messages` | 是 | 已排序的對話訊息陣列。 |
| `max_tokens` | OpenAI：選用 · Anthropic：**必需** | 最大輸出權杖數。 |
| `temperature` | 否 | 取樣溫度。OpenAI 範圍 `0`-`2`；Anthropic 範圍 `0`-`1`。 |
| `stream` | 否 | 當 `true` 時透過 SSE 串流回應。 |
| `tools` | 否 | 工具呼叫的函式/工具定義。 |

<Warning>
  協議差異：OpenAI 協議將系統提示詞置於 `messages` 中作為 `role: system`，`max_tokens` 為選用。Anthropic 協議使用頂層 `system` 欄位且**必需** `max_tokens`。
</Warning>

## 串流

設定 `stream: true` 透過 Server-Sent Events 接收回應。

- **OpenAI 協議** — 增量 `delta` 區塊，以最終 `data: [DONE]` 行終止。
- **Anthropic 協議** — 型別化 SSE 事件（`message_start`、`content_block_delta`、`message_stop` 等）；沒有 `[DONE]` 哨兵。

<CodeGroup>
```python 美國/歐洲
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.ai/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```

```python 亞太地區
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.space/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```
</CodeGroup>

## 工具呼叫

OpenAI 和 Anthropic 協議都支援工具呼叫。

```python
from openai import OpenAI

client = OpenAI(base_url="https://bearlab.ai/v1", api_key="sk-...")

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the current weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"],
        },
    },
}]

resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "What's the weather in London?"}],
    tools=tools,
)
print(resp.choices[0].message)
```

<Note>
  亞太地區使用者：改為設定 `base_url="https://bearlab.space/v1"`。
</Note>

## Gemini 原生協議

Gemini 模型可以用原生 Google AI Studio 格式呼叫，無需轉換為 OpenAI 格式。

<CodeGroup>
```python 美國/歐洲
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.ai"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```

```python 亞太地區
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.space"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```
</CodeGroup>

<Warning>
  1. 將 `base_url` 設定為地區根目錄（`https://bearlab.ai` 或 `https://bearlab.space`）；勿追加 `/v1beta`，SDK 會新增。
  2. 將 `api_version` 設定為 `v1beta`。
  3. 安裝：Python `pip install google-genai`，Node.js `npm install @google/genai`。
</Warning>

<Tip>
  重複的長前綴（固定系統提示詞、長文件）可以使用 [提示詞快取](/guides/prompt-caching) 降低重複部分的成本。
</Tip>
