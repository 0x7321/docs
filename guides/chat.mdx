對話補全是核心介面。提供 OpenAI、Anthropic、Gemini 三套相容實現，共用同一金鑰。

<Note>
  亞太區域使用者:介面地址請改用 `https://bearlab.in`，而非 `bearlab.ai`。詳見 [服務區域](/guides/availability)。
</Note>

## 端點

| 協議 | 方法與路徑 |
| --- | --- |
| OpenAI 相容 | `POST https://bearlab.ai/v1/chat/completions` |
| Anthropic 相容 | `POST https://bearlab.ai/anthropic/v1/messages` |
| Gemini 原生 | `base_url=https://bearlab.ai`，`api_version=v1beta`（Google genai SDK） |

## 基本請求

<Tabs>
  <Tab title="OpenAI 相容">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-4o",
        "messages": [
          {"role": "system", "content": "你是一個簡潔的助手。"},
          {"role": "user", "content": "你好"}
        ],
        "temperature": 0.7
      }'
    ```
  </Tab>
  <Tab title="Anthropic 相容">
    ```bash
    curl https://bearlab.ai/anthropic/v1/messages \
      -H "x-api-key: YOUR_API_KEY" \
      -H "anthropic-version: 2023-06-01" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-5",
        "max_tokens": 1024,
        "system": "你是一個簡潔的助手。",
        "messages": [{"role": "user", "content": "你好"}]
      }'
    ```
  </Tab>
</Tabs>

## 常用引數

| 引數 | 說明 |
| --- | --- |
| `model` | 模型 ID。可用列表見 [模型與定價介面](/guides/models)。 |
| `messages` | 多輪對話訊息陣列，按時間順序排列。 |
| `temperature` | 取樣溫度，`0`~`2`，越高越發散。 |
| `top_p` | 核取樣，與 `temperature` 二選一調節即可。 |
| `max_tokens` | 最大生成 token 數。Anthropic 協議下為**必填**。 |
| `stream` | 是否流式返回，`true` 時以 SSE 推送。 |
| `stop` | 停止序列。 |

<Warning>
  **協議差異**：OpenAI 協議把 system 提示放進 `messages` 的 `role: system`；Anthropic 協議用頂層 `system` 欄位，且 `max_tokens` 必填。
</Warning>

## 多輪對話

把歷史訊息按順序全部放進 `messages` 即可，服務端不儲存上下文，每次請求需自帶完整歷史。

```json
{
  "model": "gpt-4o",
  "messages": [
    {"role": "user", "content": "我叫小明"},
    {"role": "assistant", "content": "你好，小明！"},
    {"role": "user", "content": "我叫什麼？"}
  ]
}
```

## 流式輸出

設 `stream: true`，響應以 SSE 分塊推送，最後以 `data: [DONE]` 結束（OpenAI 協議）。

```python
from openai import OpenAI

client = OpenAI(api_key="YOUR_API_KEY", base_url="https://bearlab.ai/v1")

stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "寫一首四行小詩"}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```

## 響應結構（OpenAI 協議）

```json
{
  "id": "chatcmpl-...",
  "object": "chat.completion",
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {"role": "assistant", "content": "..."},
      "finish_reason": "stop"
    }
  ],
  "usage": { "prompt_tokens": 23, "completion_tokens": 58, "total_tokens": 81 }
}
```

`usage` 中的 token 數即本次計費依據，詳見 [計費與充值](/guides/pricing)。

## 工具呼叫（Function Calling）

OpenAI 與 Anthropic 協議都支援工具呼叫。

<Tabs>
  <Tab title="OpenAI 相容">
    ```python
    from openai import OpenAI

    client = OpenAI(base_url="https://bearlab.ai/v1", api_key="YOUR_API_KEY")

    tools = [{
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get the current weather for a city",
            "parameters": {
                "type": "object",
                "properties": {"city": {"type": "string"}},
                "required": ["city"],
            },
        },
    }]

    resp = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "What's the weather in London?"}],
        tools=tools,
    )
    print(resp.choices[0].message)
    ```
  </Tab>
  <Tab title="Anthropic 相容">
    ```python
    import anthropic

    client = anthropic.Anthropic(base_url="https://bearlab.ai/anthropic", api_key="YOUR_API_KEY")

    tools = [{
        "name": "get_weather",
        "description": "Get the current weather for a city",
        "input_schema": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"],
        },
    }]

    msg = client.messages.create(
        model="claude-sonnet-4-5",
        max_tokens=1024,
        tools=tools,
        messages=[{"role": "user", "content": "What's the weather in London?"}],
    )
    print(msg.content)
    ```
  </Tab>
</Tabs>

## Gemini 原生協議

Gemini 模型可直接用 Google AI Studio 的原生格式呼叫，無需轉成 OpenAI 格式，適合已在用 `genai` SDK 的專案。

```python
from google import genai
from google.genai import types

client = genai.Client(
    api_key="YOUR_API_KEY",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.ai"),
)

resp = client.models.generate_content(model="gemini-2.5-pro", contents="你好")
print(resp.text)
```

<Warning>
  1. `base_url` 設為 `https://bearlab.ai`（不帶 `/v1beta` 字尾，SDK 會自動追加）
  2. `api_version` 設為 `v1beta`
  3. 依賴：Python `pip install google-genai`、Node.js `npm install @google/genai`
</Warning>

<Tip>
  重複的長字首（固定 system 提示、長文件）可啟用 [提示詞快取](/guides/prompt-caching)，顯著降低重複部分的費用。
</Tip>
