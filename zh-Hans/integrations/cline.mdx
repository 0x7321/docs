---
title: "Cline / Roo Code"
description: "在 Cline 或 Roo Code VS Code 扩展中使用 BearLab AI。"
---

[Cline](https://github.com/cline/cline) 和其增强分支 [Roo Code](https://github.com/RooVetGit/Roo-Code) 是 VS Code AI 编程扩展。通过 OpenAI 兼容接口将它们连接到 BearLab AI。

<Steps>
  <Step title="选择提供商">
    在扩展设置中，将 API Provider 设置为 **OpenAI Compatible**。
  </Step>
  <Step title="填入连接详细信息">
    - **Base URL**：使用您所在地区的端点

      <Tabs>
        <Tab title="US / EU">
          ```
          https://bearlab.ai/v1
          ```
        </Tab>
        <Tab title="Asia-Pacific">
          ```
          https://bearlab.space/v1
          ```
        </Tab>
      </Tabs>

    - **API Key**：您的 `sk-...` 密钥
    - **Model ID**：来自 [models endpoint](/zh-Hans/guides/models) 的精确模型 id，例如 `claude-sonnet-4-6`、`gpt-5.4` 或 `deepseek-v4-pro`
  </Step>
</Steps>

<Note>
  选择 **OpenAI Compatible**，而非 Anthropic，并在 Base URL 上保留 `/v1` 后缀。Cline 在 `Authorization: Bearer sk-...` 标头中使用您的密钥调用 `POST /v1/chat/completions`。
</Note>

<Warning>
  **Model ID** 必须完全匹配 [models endpoint](/zh-Hans/guides/models) 返回的 id。名称如 `gpt-4o` 不在该列表中的并非有效的模型 id，请求将被拒绝。
</Warning>
