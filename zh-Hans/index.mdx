## 服务区域

BearLab AI面向**美国与欧盟**司法管辖区直接提供服务。

<Tip>
**亚太区域**由区域合作伙伴提供服务，该区域的 API 请求须改用专用域名 `https://bearlab.in`，而非 `https://bearlab.ai`。
</Tip>

| 区域 | API 域名 |
| --- | --- |
| 美国 / 欧盟 | `https://bearlab.ai` |
| 亚太区域 | `https://bearlab.in` |

OpenAI base_url = 域名 + `/v1`(端点 `/v1/chat/completions`);Anthropic base_url = 域名(端点 `/v1/messages`)。详见 [对话补全](/zh-Hans/guides/chat)。

详见 [服务区域](/zh-Hans/guides/availability)。

BearLab AI（BearLab.ai）把数十家厂商的对话模型收到一个 API 下。申请一个 Key、配一个 `base_url`，就能调 OpenAI、Anthropic、Google、DeepSeek 的最新对话模型，不用各家分别注册和充值。

接口同时兼容 OpenAI 和 Anthropic 两套协议，现有代码基本不用改。
