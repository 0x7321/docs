`/v1/models/pricing` 是一个**公开、免鉴权**的聚合接口，返回平台当前所有可用对话模型及其公开价格。响应严格遵循 [OpenRouter Providers 规范](https://openrouter.ai/docs/guides/community/for-providers)，可被 OpenRouter、第三方比价工具或任意自定义客户端直接读取。

<Note>
  亚太区域用户:请改用专用域名 `https://bearlab.in`，而非 `https://bearlab.ai`。详见 [服务区域](/zh-Hans/guides/availability)。
</Note>

<Note>
  这是「公开价格」入口，不携带用户身份。返回的是匿名上下文下 `default` 分组的价格。要查询某个已登录用户的实际成交价（分组、等级折扣等），请用控制台内的定价页。
</Note>

## 何时使用

| 场景 | 说明 |
| --- | --- |
| 第三方比价 | OpenRouter 等聚合器抓取模型目录与公开价格 |
| 客户端展示 | App、机器人、SDK 需要渲染模型列表 |
| 监控 | 跟踪模型目录或价格随时间的变化 |

## 端点

| 项 | 值 |
| --- | --- |
| **URL** | `https://bearlab.ai/v1/models/pricing` |
| **方法** | `GET` |
| **鉴权** | 无（公开） |
| **限流** | 严格的单 IP 关键接口限流 |
| **缓存** | 60 秒进程内缓存；响应带 `Cache-Control: public, max-age=60` |

### 请求示例

<CodeGroup>
```bash cURL
curl https://bearlab.ai/v1/models/pricing
```

```python Python
import requests

resp = requests.get("https://bearlab.ai/v1/models/pricing", timeout=10)
data = resp.json()["data"]
for m in data[:3]:
    print(m["id"], m["pricing"])
```

```javascript Node.js
const resp = await fetch("https://bearlab.ai/v1/models/pricing");
const { data } = await resp.json();
console.log(data.slice(0, 3).map(m => ({ id: m.id, pricing: m.pricing })));
```
</CodeGroup>

```json

      "output_modalities": ["text"],
      "context_length": 128000,
      "max_output_length": 16384,
      "pricing": {
        "prompt": "0.0000025",
        "completion": "0.00001",

        "input_cache_read": "0.00000125"
      },
      "supported_sampling_parameters": [
        "temperature", "top_p", "frequency_penalty", "presence_penalty",
```

### 模型字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 模型唯一标识，调用 `/v1/chat/completions` 时填这个值 |
| `name` | `string` | 展示名（如 `GPT-4o`），未配置时回退为 `id` |
| `created` | `int64` | 模型发布时间（Unix 秒） |
| `input_modalities` / `output_modalities` | `string[]` | 输入/输出模态，至少包含 `text` |
| `context_length` | `int` | 最大上下文窗口（token） |
| `max_output_length` | `int` | 单次最大输出 token 数 |
| `pricing` | `Pricing` | 价格对象，见下 |
| `pricing_tiers` | `Pricing[]` | 阶梯价（最多一项；按上下文长度阈值触发） |
| `supported_sampling_parameters` | `string[]` | 支持的采样参数 |
| `supported_features` | `string[]` | 支持的能力标记，见下 |

### 价格字段

价格均为 **USD 字符串**，token 类按 token 计价。

| 字段 | 单位 | 说明 |
| --- | --- | --- |
| `prompt` | USD / 输入 token | 输入单价 |
| `completion` | USD / 输出 token | 输出单价 |
| `request` | USD / 次 | 每次请求的固定费（多数对话模型为 `"0"`） |
| `internal_reasoning` | USD / 推理 token | 推理 token 价（o 系 / Claude thinking） |
| `input_cache_read` | USD / token | 命中缓存的读取价（通常更低） |
| `input_cache_write` | USD / token | 缓存写入价 |
| `web_search` | USD / 次 | 内置联网搜索工具费 |

<Note>
  值为 `"0"` 可能表示「免费」或「该模型不适用此维度」。请结合 `supported_features` 与 `supported_sampling_parameters` 判断能力，不要只看价格。
</Note>

### `supported_features` 取值

| 值 | 含义 |
| --- | --- |
| `tools` | 函数调用 / 工具使用 |
| `reasoning` | 返回推理 token（`reasoning_content` / `thinking`） |
| `json_mode` | 支持 JSON 模式输出 |
| `structured_outputs` | 支持 schema 约束的结构化输出 |
| `logprobs` | 返回 token 对数概率 |
| `web_search` | 内置联网搜索工具 |

## 阶梯价

部分模型在超过某个上下文长度阈值后改用另一档单价（如 Gemini 2.5 Pro 在 ≥128K token 时翻倍）。上层档位通过 `pricing_tiers` 暴露：`pricing` 是基础档，`pricing_tiers[0]` 是上层档，当总输入 token ≥ `min_context` 时整体套用上层档价格（取满足条件中阈值最高的一档，非累进）。

## 缓存与更新节奏

| 方面 | 行为 |
| --- | --- |
| 服务端缓存 | 60 秒进程内缓存，后台改价 1 分钟内生效 |
| 客户端缓存 | `Cache-Control: public, max-age=60`，CDN / 浏览器可安全缓存 |
| 建议轮询 | 最多每 5 分钟一次，激进轮询会被限流 |

## 与其它接口的区别

| 接口 | 用途 | 鉴权 |
| --- | --- | --- |
| `/v1/models/pricing` | 本接口：OpenRouter 格式的公开价格 | 公开 |
| `/v1/models` | OpenAI SDK 兼容的模型 id 列表 | 需令牌 |
