三个步骤调用模型：创建密钥、将您的客户端指向正确的基础 URL，然后发送您的首个聊天请求。

<Note>
  根据您的地区选择基础 URL。**美国 / 欧洲：** `https://bearlab.ai`。**亚太地区：** `https://bearlab.space`。其他所有内容（密钥、模型、请求格式）在各地区都是相同的。请参阅[服务可用性](/zh-Hans/guides/availability)。
</Note>

<Steps>
  <Step title="注册并充值">
    在[控制台](https://bearlab.ai)注册并在钱包页面上添加资金。新用户可以先充值少量金额进行测试。
  </Step>
  <Step title="创建 API 密钥">
    进入仪表板，选择 API 令牌，点击添加令牌，并复制 `sk-...` 密钥。密钥仅会完整显示一次。您在每个请求上将其作为 `Authorization: Bearer sk-...` 传递。
  </Step>
  <Step title="设置您的基础 URL">
    使用与您的地区和协议相符的基础 URL：

    | 协议 | 美国 / 欧洲 | 亚太地区 |
    | --- | --- | --- |
    | OpenAI 兼容 | `https://bearlab.ai/v1` | `https://bearlab.space/v1` |
    | Anthropic | `https://bearlab.ai/v1/messages` | `https://bearlab.space/v1/messages` |
  </Step>
  <Step title="进行您的首个请求">
    发送下面的示例。将 `sk-...` 替换为您刚创建的密钥。
  </Step>
</Steps>

## 示例

OpenAI 兼容端点是 `POST /v1/chat/completions`；Anthropic 端点是 `POST /v1/messages`。两者都使用 `Authorization: Bearer sk-...` 进行身份验证。`model` 值必须是[模型列表](/zh-Hans/guides/models)为您的密钥返回的值 — 下面的示例使用此帐户可用的模型。

### OpenAI 兼容

<CodeGroup>
```bash US / EU (bearlab.ai)
curl https://bearlab.ai/v1/chat/completions \
  -H "Authorization: Bearer sk-..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
  }'
```

```bash Asia-Pacific (bearlab.space)
curl https://bearlab.space/v1/chat/completions \
  -H "Authorization: Bearer sk-..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
  }'
```
</CodeGroup>

### Anthropic

Anthropic 端点需要 `max_tokens` 和 `anthropic-version` 标头。您可以使用 `Authorization: Bearer sk-...` 进行身份验证（网关也接受 `x-api-key: sk-...`）。

<CodeGroup>
```bash US / EU (bearlab.ai)
curl https://bearlab.ai/v1/messages \
  -H "Authorization: Bearer sk-..." \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 256,
    "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
  }'
```

```bash Asia-Pacific (bearlab.space)
curl https://bearlab.space/v1/messages \
  -H "Authorization: Bearer sk-..." \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 256,
    "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
  }'
```
</CodeGroup>

### Python（OpenAI SDK）

将 SDK 指向您地区的 `/v1` 基础 URL。SDK 会自动附加 `/chat/completions`。

<CodeGroup>
```python US / EU (bearlab.ai)
from openai import OpenAI

client = OpenAI(
    api_key="sk-...",
    base_url="https://bearlab.ai/v1",
)

resp = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Introduce yourself in one sentence"}],
)
print(resp.choices[0].message.content)
```

```python Asia-Pacific (bearlab.space)
from openai import OpenAI

client = OpenAI(
    api_key="sk-...",
    base_url="https://bearlab.space/v1",
)

resp = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Introduce yourself in one sentence"}],
)
print(resp.choices[0].message.content)
```
</CodeGroup>
