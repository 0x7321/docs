title: "快速开始"
description: "5 分钟内创建密钥并发出第一个对话请求。"
---

只需三步即可调用模型。

<Note>
  亚太区域用户:API 接口地址请改用 `https://bearlab.in`（`/v1`、`/anthropic`），而非 `bearlab.ai`。详见 [服务区域](/zh-Hans/guides/availability)。
</Note>

<Steps>
  <Step title="注册并充值">
    前往 [控制台](https://bearlab.ai) 注册账号，在「钱包」页完成充值。新用户可先小额充值测试。
  </Step>
  <Step title="创建 API Key">
    进入「令牌 / API Keys」页，点击「新建」，复制生成的 `sk-...` 密钥并妥善保管。密钥只在创建时完整显示一次。
  </Step>
  <Step title="发起第一个请求">
    把下面示例里的 `YOUR_API_KEY` 换成你的密钥即可。
  </Step>
</Steps>

## 调用示例

<Tabs>
  <Tab title="OpenAI 兼容">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gpt-4o",
        "messages": [{"role": "user", "content": "用一句话介绍你自己"}]
      }'
    ```
  </Tab>
  <Tab title="Anthropic 兼容">
    ```bash
    curl https://bearlab.ai/anthropic/v1/messages \
      -H "x-api-key: YOUR_API_KEY" \
      -H "anthropic-version: 2023-06-01" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-5",
        "max_tokens": 256,
        "messages": [{"role": "user", "content": "用一句话介绍你自己"}]
      }'
    ```
  </Tab>
  <Tab title="Python (OpenAI SDK)">
    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://bearlab.ai/v1",
    )

    resp = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "用一句话介绍你自己"}],
    )
    print(resp.choices[0].message.content)
    ```
  </Tab>
</Tabs>
