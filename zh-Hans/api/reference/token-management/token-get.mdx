## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非 `sk-` API Key） |

接口：`GET /api/token/`

分页列出当前账号的所有 API Key。名称 / 前缀搜索请用 `GET /api/token/search`。

## 查询参数

| 查询参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `p` |  | integer | 页码，从 1 开始 |
| `page_size` |  | integer | 每页数量 |

```bash
curl "https://bearlab.ai/api/token/?p=1&page_size=10" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json 回应
{
  "success": true,
  "message": "",
  "data": {
    "items": [
      {
        "id": 10623,
        "user_id": 42,
        "key": "ntEH**********7nAh",
        "status": 1,
        "name": "my-token",
        "created_time": 1781977779,
        "accessed_time": 1781977779,
        "expired_time": -1,
        "remain_quota": 0,
        "unlimited_quota": true,
        "used_quota": 0,
        "model_limits_enabled": false,
        "model_limits": "",
        "allow_ips": "",
        "group": "",
        "cross_group_retry": false
      }
    ],
    "page": 1,
    "page_size": 10,
    "total": 7
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `key` | string | 已脱敏（首 4 + `**********` + 末 4 位）；完整 Key 仅创建时一次性可见 |
| `status` | integer | `1` 启用 / `2` 禁用 / `3` 过期 / `4` 耗尽 |
| `expired_time` | integer | 过期时间（Unix 秒），`-1` 永不过期 |
| `remain_quota` | integer | 剩余额度（quota，500000 = 1 美元） |
| `unlimited_quota` | boolean | 是否无限额度 |
| `used_quota` | integer | 已用额度 |
| `model_limits_enabled` | boolean | 是否启用模型白名单 |
| `model_limits` | string | 模型白名单（逗号分隔，空串不限制） |
| `allow_ips` | string \| null | IP 白名单（空串 / `null` 不限制） |
| `group` | string | 令牌分组 |
| `cross_group_retry` | boolean | 是否跨分组重试（仅 `auto` 分组有效） |
| `data.page` / `page_size` / `total` | integer | 分页信息 |

<Note>
来源：new-api 源码 `controller/token.go` `GetAllTokens` + `model/token.go` `Token`。`GET /api/token/` 仅接受分页参数；`key` 经 `MaskTokenKey` 脱敏。旧文档的 `remain_count` / `unlimited_count` / `exclude_ips` / `rate_limits` 等字段在 new-api 中**不存在**。
</Note>
