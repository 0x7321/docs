接口：`GET /api/usage/token/`

## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_MODEL_API_KEY`，令牌管理创建出来的模型 API Key；不要用账号访问令牌 `YOUR_ACCESS_TOKEN` |

查詢該 API Key 自身的額度與已用量。此端點使用 API Key 本身鑑權，不是帳號 `access_token`。

<Warning>
必须使用令牌管理里创建出来的模型 API Key（本文统一写作 `YOUR_MODEL_API_KEY`，通常以 `sk-...` 形式用于模型调用）。这里不能用账号访问令牌 `YOUR_ACCESS_TOKEN`；用错会返回 `401 Invalid Token`。路径须带尾部斜杠。
</Warning>

```bash
curl https://bearlab.ai/api/usage/token/ \
  -H "Authorization: Bearer YOUR_MODEL_API_KEY"
```

```json Response
{
  "code": true,
  "message": "ok",
  "data": {
    "object": "token_usage",
    "name": "default",
    "total_granted": 500000,
    "total_used": 444364,
    "total_available": 55636,
    "unlimited_quota": false,
    "model_limits": {},
    "model_limits_enabled": false,
    "expires_at": 0
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `code` | boolean | 成功为 `true`（注意此接口外层用 `code`，**不是** `success`） |
| `data.object` | string | 固定为 `token_usage` |
| `data.name` | string | 令牌名称 |
| `data.total_granted` | integer | 总额度（剩余 + 已用，quota） |
| `data.total_used` | integer | 已用额度 |
| `data.total_available` | integer | 剩余额度 |
| `data.unlimited_quota` | boolean | 是否为无限额度令牌 |
| `data.model_limits` | object | 模型白名单映射（未启用时为空） |
| `data.model_limits_enabled` | boolean | 是否启用模型白名单 |
| `data.expires_at` | integer | 过期时间（Unix 秒，`0` 表示永不过期） |

<Note>
来源：new-api 源码 `controller/token.go` `GetTokenUsage`。成功响应外层用 `{ code: true, message: "ok", data }`（与其他接口的 `success` 包装不同）。鉴权失败返回 `401 Invalid Token`。
</Note>
