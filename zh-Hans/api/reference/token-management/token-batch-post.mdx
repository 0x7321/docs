---
title: "批量删除"
description: "POST /api/token/batch"
---

## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非 `sk-` API Key） |
| `Content-Type` | ✓ | `application/json` |

接口：`POST /api/token/batch`

一次删除多个 API Key（`DeleteTokenBatch`，批量删除，非创建）。

## 请求体

| 字段 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `ids` | ✓ | integer[] | 要删除的 Token ID 列表，ID 来自 `GET /api/token/` 的 `data.items[].id` |

```bash
curl -X POST https://bearlab.ai/api/token/batch \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"ids": [10623, 10624]}'
```

<Note>
前端降级策略：批量删除失败时会逐个调用 `DELETE /api/token/{id}`。
</Note>

```json Response
{ "success": true, "message": "", "data": 2 }
```


<Note>
来源：new-api 源码 `controller/token.go` `DeleteTokenBatch`。请求体 `{ ids: [...] }`（至少 1 个），响应 `data` 为成功删除的数量。
</Note>
