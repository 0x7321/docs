## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非令牌管理生成的 `sk-` API Key） |

接口：`GET /api/data/self`

返回当前用户在指定时间范围内、按「日期 × 模型」聚合的用量明细。每条记录对应一天某个模型的调用次数、消耗 token 与额度。

## 查询参数

| 参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `start_timestamp` |  | integer | 起始时间，Unix 秒 |
| `end_timestamp` |  | integer | 结束时间，Unix 秒 |

```bash
curl "https://bearlab.ai/api/data/self?start_timestamp=1717200000&end_timestamp=1718000000" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json 回应
{
  "success": true,
  "message": "",
  "data": [
    {
      "id": 1,
      "user_id": 42,
      "username": "alice",
      "model_name": "gpt-5.4",
      "created_at": 1717200000,
      "use_group": "default",
      "token_id": 7,
      "channel_id": 12,
      "node_name": "",
      "token_used": 1024,
      "count": 8,
      "quota": 240
    }
  ]
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `data` | array | 用量记录数组（`data` 直接是数组，无分页包装） |
| `model_name` | string | 模型名 |
| `created_at` | integer | 聚合日期（Unix 秒） |
| `use_group` | string | 使用的分组 |
| `token_id` | integer | 令牌 ID |
| `channel_id` | integer | 渠道 ID |
| `token_used` | integer | 消耗 token 数 |
| `count` | integer | 调用次数 |
| `quota` | integer | 消耗额度（quota，500000 = 1 美元） |

<Note>
来源：new-api 源码 `controller/usedata.go` `GetUserQuotaDates` + `model/usedata.go` `QuotaData`。服务端仅读取 `start_timestamp` / `end_timestamp`，无 `granularity` / `dimension` 参数；响应为 `{ success, message, data: [...] }`，`data` 是扁平数组。
</Note>
