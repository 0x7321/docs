---
title: "获取个人日志统计"
description: "GET /api/log/self/stat"
---

## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非 `sk-` API Key） |

接口：`GET /api/log/self/stat`

获取当前账号的调用统计（消耗额度、RPM、TPM）。

## 查询参数

| 查询参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `start_timestamp` |  | integer | 开始时间，Unix 秒 |
| `end_timestamp` |  | integer | 结束时间，Unix 秒 |
| `type` |  | integer | 日志类型 |
| `token_name` |  | string | 按令牌名称筛选 |
| `model_name` |  | string | 按模型名筛选 |
| `channel` |  | integer | 按渠道 ID 筛选 |
| `group` |  | string | 按分组筛选 |

```bash
curl "https://bearlab.ai/api/log/self/stat" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json Response
{
  "success": true,
  "message": "",
  "data": {
    "quota": 0,
    "rpm": 0,
    "tpm": 0
  }
}
```

| 字段 | 说明 |
| --- | --- |
| `quota` | 消耗额度（quota，500000 = 1 美元） |
| `rpm` | 每分钟请求数 |
| `tpm` | 每分钟 Token 数 |

<Note>
来源：new-api 源码 `controller/log.go` `GetLogsSelfStat` + `model/log.go` `Stat`。`data` 仅含 `quota` / `rpm` / `tpm` 三个字段（旧文档的 `mpm` / `unfinished_count` / `unfinished_by_vendor` 在 new-api 中不存在）。
</Note>
