[Codex CLI](https://github.com/openai/codex) is OpenAI's official command-line coding agent, connected through a custom provider.

## Configuration

Edit `~/.codex/config.toml` and add a provider pointing at BearLab:

```toml
model = "gpt-4o"
model_provider = "bearlab"

# Third-party providers must disable response storage
disable_response_storage = true

[model_providers.bearlab]
name = "BearLab"
base_url = "https://bearlab.ai/v1"
env_key = "BEARLAB_API_KEY"
wire_api = "responses"
```

Then export the key and run:

```bash
export BEARLAB_API_KEY="YOUR_API_KEY"
codex
```

<Warning>
  Use the OpenAI-compatible address (`/v1`). Third-party providers must set `wire_api = "responses"` (the only protocol Codex supports) and `disable_response_storage = true`, or some features error. `model` must exist in the [models endpoint](/en/guides/models).
</Warning>
