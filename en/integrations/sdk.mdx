---
title: "Official SDKs (Python / Node)"
description: "Use the official OpenAI or Anthropic SDKs with BearLab."
---

Official SDKs only need a changed `base_url` and key to point at BearLab.

## OpenAI SDK

<CodeGroup>
```python Python
from openai import OpenAI

client = OpenAI(api_key="YOUR_API_KEY", base_url="https://bearlab.ai/v1")

resp = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello"}],
)
print(resp.choices[0].message.content)
```

```javascript Node.js
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "YOUR_API_KEY",
  baseURL: "https://bearlab.ai/v1",
});

const resp = await client.chat.completions.create({
  model: "gpt-4o",
  messages: [{ role: "user", content: "Hello" }],
});
console.log(resp.choices[0].message.content);
```
</CodeGroup>

## Anthropic SDK

<CodeGroup>
```python Python
from anthropic import Anthropic

client = Anthropic(api_key="YOUR_API_KEY", base_url="https://bearlab.ai/anthropic")

msg = client.messages.create(
    model="claude-sonnet-4-5",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}],
)
print(msg.content[0].text)
```

```javascript Node.js
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: "YOUR_API_KEY",
  baseURL: "https://bearlab.ai/anthropic",
});

const msg = await client.messages.create({
  model: "claude-sonnet-4-5",
  max_tokens: 1024,
  messages: [{ role: "user", content: "Hello" }],
});
console.log(msg.content[0].text);
```
</CodeGroup>

<Tip>
  Any tool that supports a custom OpenAI base URL (LobeChat, Chatbox, Continue, Zed, Immersive Translate) works the same way: set the base URL to `https://bearlab.ai/v1` and use your key.
</Tip>
