[Claude Code](https://docs.claude.com/claude-code) is Anthropic's official command-line coding tool, connected through the Anthropic-compatible protocol.

## Configuration

Point environment variables at the Anthropic-compatible endpoint:

```bash
export ANTHROPIC_BASE_URL="https://bearlab.ai/anthropic"
export ANTHROPIC_AUTH_TOKEN="YOUR_API_KEY"

claude
```

- `ANTHROPIC_BASE_URL`: the Anthropic-compatible address (note `/anthropic`, no `/v1`).
- `ANTHROPIC_AUTH_TOKEN`: your API key.

## Persist it

Add the two lines to `~/.zshrc` or `~/.bashrc`:

```bash
echo 'export ANTHROPIC_BASE_URL="https://bearlab.ai/anthropic"' >> ~/.zshrc
echo 'export ANTHROPIC_AUTH_TOKEN="YOUR_API_KEY"' >> ~/.zshrc
```

<Warning>
  Use a Claude model (e.g. `claude-sonnet-4-5`) that exists in the [models endpoint](/en/guides/models). If auth fails, confirm you used `ANTHROPIC_AUTH_TOKEN`, not `ANTHROPIC_API_KEY`.
</Warning>
