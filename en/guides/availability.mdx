BearLab (bearlab.ai) provides services directly in the **United States and European Union** jurisdictions.

### Asia-Pacific

The Asia-Pacific region is served by a **Regional Partner based in India**.

API requests from this region must use the dedicated domain `https://bearlab.in` instead of `https://bearlab.ai`.

Policies for service in this region are set by the Regional Partner, and users are responsible for compliance with applicable local laws.

| Region | API domain | OpenAI | Anthropic |
| --- | --- | --- | --- |
| US / EU | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| Asia-Pacific | `https://bearlab.in` | `https://bearlab.in/v1/chat/completions` | `https://bearlab.in/v1/messages` |
