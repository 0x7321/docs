title: "Service Availability"
description: "Where BearLab serves directly, and how the Asia-Pacific region connects."
---

BearLab (bearlab.ai) provides services directly in the **United States and European Union** jurisdictions.

The Asia-Pacific region is served by a **Regional Partner**. API requests from this region must use the dedicated domain **`https://bearlab.in`** instead of `https://bearlab.ai`. Policies for service in this region are set by the Regional Partner. Users in the Asia-Pacific region are responsible for compliance with applicable local laws.

| Region | API domain |
| --- | --- |
| US / EU | `https://bearlab.ai` |
| Asia-Pacific | `https://bearlab.in` |

OpenAI base_url = domain + `/v1` (endpoint `/v1/chat/completions`); Anthropic base_url = domain (endpoint `/v1/messages`). See [Chat Completions](/en/guides/chat).
