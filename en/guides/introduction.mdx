> BearLab (BearLab.ai) brings conversational models from dozens of leading providers under one API. OpenAI, Anthropic, Google, and DeepSeek are all reachable with a single key.

## What is BearLab

BearLab is a conversational-model aggregation platform. Instead of registering, topping up, and managing a separate key with OpenAI, Anthropic, and Google individually, you create one key here, set one `base_url`, and call their latest conversational models.

The interface is compatible with both the OpenAI and Anthropic protocols, so existing code usually works after changing the endpoint and key.

We do one thing, conversational models, which keeps the interface simple, billing clear, and the service stable.

## Key capabilities

**One key for every model**
Different vendors and models through one unified interface. One account for access, one bill for spend, one place to see usage.

**OpenAI / Anthropic dual protocol**
Both protocols are natively supported. Cursor, Claude Code, Cline, and Codex connect by setting the endpoint, with no need to care which vendor is behind a model.

**Stable and available**
Popular models are backed by multiple providers. If one fails, requests fail over to another automatically.

**Pay as you go**
Pay only for what you use. No monthly fee, no minimum top-up. Prices track the upstream providers, and balance is visible in real time.

**Observable usage**
Every call records its request, response, and token consumption, making it easy to troubleshoot and account for cost.

## Conversational models only

BearLab does not offer image, audio, video, music, or embedding models. We focus on making conversational models stable and complete. For those other capabilities, use a dedicated platform.
