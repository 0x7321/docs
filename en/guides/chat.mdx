Chat completion is the core endpoint, offered through three compatible interfaces (OpenAI, Anthropic, Gemini) that share one key.

<Note>
  BearLab AI serves two regions. US / EU users call `https://bearlab.ai`; Asia-Pacific users call the dedicated domain `https://bearlab.space`. The two regions share the same key and model catalog. See [Service Availability](/en/guides/availability).
</Note>

## Endpoints

| Protocol | Method and path (US / EU) | Method and path (Asia-Pacific) |
| --- | --- | --- |
| OpenAI-compatible | `POST https://bearlab.ai/v1/chat/completions` | `POST https://bearlab.space/v1/chat/completions` |
| Anthropic-compatible | `POST https://bearlab.ai/v1/messages` | `POST https://bearlab.space/v1/messages` |
| Gemini native | `base_url=https://bearlab.ai`, `api_version=v1beta` (Google genai SDK) | `base_url=https://bearlab.space`, `api_version=v1beta` |

Authenticate every request with your API key (`sk-...`) in the `Authorization: Bearer` header. The Anthropic protocol additionally accepts the `x-api-key` header.

## Common parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `model` | Yes | Model ID. See [Models & Pricing API](/en/guides/models). |
| `messages` | Yes | Ordered array of conversation messages. |
| `max_tokens` | OpenAI: optional · Anthropic: **required** | Maximum output tokens. |
| `temperature` | No | Sampling temperature. OpenAI range `0`-`2`; Anthropic range `0`-`1`. |
| `stream` | No | Stream the response over SSE when `true`. |
| `tools` | No | Function / tool definitions for tool calling. |

<Warning>
  Protocol difference: the OpenAI protocol puts the system prompt in `messages` as `role: system`, and `max_tokens` is optional. The Anthropic protocol uses a top-level `system` field and **requires** `max_tokens`.
</Warning>

## Streaming

Set `stream: true` to receive the response over Server-Sent Events.

- **OpenAI protocol** — incremental `delta` chunks, terminated by a final `data: [DONE]` line.
- **Anthropic protocol** — typed SSE events (`message_start`, `content_block_delta`, `message_stop`, …); there is no `[DONE]` sentinel.

<CodeGroup>
```python US / EU
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.ai/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```

```python Asia-Pacific

client = OpenAI(api_key="sk-...", base_url="https://bearlab.space/v1")
```

## Tool calling

Both the OpenAI and Anthropic protocols support tool calling.

```python
from openai import OpenAI

client = OpenAI(base_url="https://bearlab.ai/v1", api_key="sk-...")

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the current weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"],
        },
    },
}]

resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "What's the weather in London?"}],
    tools=tools,
)
print(resp.choices[0].message)
```

<Note>
  Asia-Pacific users: set `base_url="https://bearlab.space/v1"` instead.
</Note>

## Gemini native protocol

Gemini models can be called in the native Google AI Studio format, with no conversion to OpenAI format.

<CodeGroup>
```python US / EU
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.ai"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```

```python Asia-Pacific
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.space"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```
</CodeGroup>

<Warning>
  1. Set `base_url` to the region root (`https://bearlab.ai` or `https://bearlab.space`); do not append `/v1beta`, the SDK adds it.
  2. Set `api_version` to `v1beta`.
  3. Install: Python `pip install google-genai`, Node.js `npm install @google/genai`.
</Warning>

<Tip>
  Long repeated prefixes (fixed system prompts, long documents) can use [Prompt Caching](/en/guides/prompt-caching) to lower the cost of the repeated part.
</Tip>
