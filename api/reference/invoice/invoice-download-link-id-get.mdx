接口：`GET /api/invoice/download-link/{id}`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 發票 ID |

```bash
curl https://bearlab.ai/api/invoice/download-link/123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": { "url": "https://..." },
  "success": true
}
```
