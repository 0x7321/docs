---
title: "提交發票申請"
description: "POST /api/invoice/submit"
---

接口：`POST /api/invoice/submit`

```bash
curl -X POST https://bearlab.ai/api/invoice/submit \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "order_ids": [6701],
    "company_name": "範例科技有限公司",
    "tax_number": "91310000XXXXXXXX",
    "email": "billing@example.com"
  }'
```

| 欄位 | 必填 | 說明 |
| --- | --- | --- |
| `order_ids` | ✓ | 訂單 ID 陣列，來自可開票訂單 |
| `company_name` | ✓ | 發票抬頭 |
| `tax_number` | ✓ | 稅號 |
| `email` | ✓ | 發票接收郵件 |
