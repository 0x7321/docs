接口：`GET /api/invoice/latest_info`

返回上次提交的發票抬頭資訊。首次申請前為 `null`。

```bash
curl https://bearlab.ai/api/invoice/latest_info \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應（無歷史記錄）
{ "data": null, "success": true }
```

```json 回應（有歷史記錄）
{
  "data": {
    "company_name": "範例科技有限公司",
    "tax_number": "91310000XXXXXXXX",
    "email": "billing@example.com"
  },
  "success": true
}
```
