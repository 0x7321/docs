接口：`GET /api/auto-recharge/settings`

<Warning>
路徑使用連字符 `auto-recharge`。下劃線版本 `/api/auto_recharge/self` 返回 404。
</Warning>

```bash
curl https://bearlab.ai/api/auto-recharge/settings \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "enabled": false,
    "status": "not_configured"
  },
  "message": "",
  "success": true
}
```

完整啟用後的欄位：`enabled` / `status` / `threshold_amount` / `target_amount` / `gateway_uuid` / `payment_method_label`。
