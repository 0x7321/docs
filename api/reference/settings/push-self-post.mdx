接口：`POST /api/push/self`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `notice_type` | ✓ | 通知方式，如 `email` |
| `notice_email` |  | 通知郵箱 |
| `webhook_url` |  | Webhook 地址 |
| `webhook_token` |  | Webhook token |
| `tip_quota` |  | 餘額提醒閾值 |

```bash
curl -X POST https://bearlab.ai/api/push/self \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "notice_type": "email",
    "notice_email": "user@example.com",
    "webhook_url": "",
    "webhook_token": "",
    "tip_quota": "10"
  }'
```
