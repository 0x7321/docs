接口：`PUT /api/user/self`

更新顯示名稱、語言、頭像等基本設定。

```bash
curl -X PUT https://bearlab.ai/api/user/self \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"display_name":"My Name","language":"zh-Hant"}'
```
