---
title: "删除 Token"
description: "DELETE /api/token/{id}"
---

## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非 `sk-` API Key） |

接口：`DELETE /api/token/{id}`

删除指定的 API Key。

## 路径参数

| 参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `id` | ✓ | integer | 要删除的 Token ID，来自 `GET /api/token/` 的 `data.items[].id` |

<Warning>
请用**无尾斜杠**路径 `DELETE /api/token/{id}`。带尾斜杠 `DELETE /api/token/{id}/` 会被 gin 以 307 重定向到无尾斜杠路径；客户端若不跟随 307 则不会删除。
</Warning>

```bash
curl -X DELETE https://bearlab.ai/api/token/10623 \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```json Response
{
  "success": true,
  "message": ""
}
```

<Note>
来源：new-api 源码 `controller/token.go` `DeleteToken`。成功响应仅 `{ success, message }`，无 `data` 字段。
</Note>
