---
title: "开始注册 Passkey"
description: "POST /api/user/passkey/register/begin"
---

接口：`POST /api/user/passkey/register/begin`

```bash
curl -X POST https://bearlab.ai/api/user/passkey/register/begin \
  -H "Authorization: Bearer YOUR_API_KEY"
```

<Note>
本实例目前返回 Passkey 登录未启用，表示路由存在但平台设置未开启。
</Note>
