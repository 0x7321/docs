---
title: "完成验证 Passkey"
description: "POST /api/user/passkey/verify/finish"
---

接口：`POST /api/user/passkey/verify/finish`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `credential` | ✓ | WebAuthn 验证凭证响应对象 |

```bash
curl -X POST https://bearlab.ai/api/user/passkey/verify/finish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"credential":{}}'
```
