---
title: "KYC 认证状态"
description: "GET /api/user/verify/self"
---

接口：`GET /api/user/verify/self`

```bash
curl https://bearlab.ai/api/user/verify/self \
  -H "Authorization: Bearer YOUR_API_KEY"
```

| `type` 值 | 说明 |
| --- | --- |
| `""` | 未认证 |
| `"personal"` | 个人认证 |
| `"enterprise"` | 企业认证 |
| `"passport"` | 护照认证 |

| `status` 值 | 说明 |
| --- | --- |
| `""` | 未提交 |
| `"PROCESSING"` | 审核中 |
| `"SUCCESS"` | 已通过 |
| `"FAIL"` | 已拒绝 |
