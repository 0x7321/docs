## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌，形如 `g+...==`，不是令牌管理生成的模型 API Key |

接口：`GET /api/data/self`

<Warning>
此接口是管理端用量统计，必须使用账号访问令牌 `YOUR_ACCESS_TOKEN`（形如 `g+...==`）。不要使用令牌管理生成的模型 API Key `YOUR_MODEL_API_KEY`。实测无鉴权返回 401，用账号访问令牌返回 200。
</Warning>

<Warning>
**RixAPI 專屬格式**：回應頂層無 `data` 包裝，`statistics` / `time_series` / `model_distribution` 直接在根物件。舊版 new-api 返回 `{ "data": [] }` 扁平陣列，格式完全不同。
</Warning>

| 查询参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `start_timestamp` |  | integer | 开始时间，Unix 秒 |
| `end_timestamp` |  | integer | 结束时间，Unix 秒 |
| `granularity` |  | string | 聚合粒度，实测可用 `day`；也支持 `hour` |
| `dimension` |  | string | 统计维度，例如 `model` |

```bash
curl "https://bearlab.ai/api/data/self?granularity=day" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
