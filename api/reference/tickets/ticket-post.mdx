接口：`POST /api/ticket`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `subject` | ✓ | 工單標題 |
| `content` | ✓ | 工單內容 |

```bash
curl -X POST https://bearlab.ai/api/ticket \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "問題標題",
    "content": "請描述您遇到的問題..."
  }'
```
