title: "回覆工單"
description: "POST /api/ticket/{id}/reply"
---

接口：`POST /api/ticket/{id}/reply`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `content` | ✓ | 回覆內容 |

```bash
curl -X POST https://bearlab.ai/api/ticket/123/reply \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "補充說明..."}'
```
