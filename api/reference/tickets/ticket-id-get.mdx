接口：`GET /api/ticket/{id}`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

```bash
curl https://bearlab.ai/api/ticket/123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應（推斷）
{
  "data": {
    "id": 123,
    "subject": "問題標題",
    "status": "open",
    "content": "問題描述",
    "replies": [
      {
        "id": 1,
        "content": "回覆內容",
        "created_at": 1781977779
      }
    ]
  }
}
```
