title: "更新工單狀態"
description: "PUT /api/ticket/{id}/status"
---

接口：`PUT /api/ticket/{id}/status`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `status` | ✓ | 目標狀態 |

```bash
curl -X PUT https://bearlab.ai/api/ticket/123/status \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"status": "closed"}'
```

| `status` 值 | 說明 |
| --- | --- |
| `open` | 開啟中 |
| `in_progress` | 處理中 |
| `closed` | 已關閉 |
