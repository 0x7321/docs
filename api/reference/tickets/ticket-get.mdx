接口：`GET /api/ticket/`

<Warning>
路徑必須帶**尾部斜杠** `/api/ticket/`。不帶斜杠時伺服器返回 **301 重定向**，curl 需加 `-L` 跟隨，否則拿不到資料。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `page` | 頁碼 |
| `size` | 每頁數量（注意：不是 `page_size`） |

```bash
curl -L "https://bearlab.ai/api/ticket/?page=1&size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "data": [],
    "page": 1,
    "size": 10,
    "total_count": 0
  }
}
```

<Note>
外層的 `data` 是標準 API 包裝，內層的 `data` 是工單陣列。這是雙層 `data` 結構。實測確認 `size` 欄位由查詢參數決定（預設 10）。
</Note>
