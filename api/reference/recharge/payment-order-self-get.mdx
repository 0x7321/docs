接口：`GET /api/payment/order/self`

| 参数 | 说明 |
| --- | --- |
| `page` | 页码 |
| `page_size` | 每页数量 |

```bash
curl "https://bearlab.ai/api/payment/order/self?page=1&page_size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

| `status` | 说明 |
| --- | --- |
| `pending` | 待支付 |
| `paid` | 已支付 |
| `closed` | 已关闭（已付款订单完成后） |
| `cancelled` | 已取消 |
| `refunded` | 已退款 |
