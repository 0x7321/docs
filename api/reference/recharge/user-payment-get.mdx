接口：`GET /api/user/payment`

列出当前用户可用的付款方式。

```bash
curl https://bearlab.ai/api/user/payment \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": [
    {
      "type": "epay",
      "uuid": "98de8f0bc169...",
      "name": "微信",
      "icon": "wxpay",
      "min_amount": "10",
      "max_amount": "10000",
      "currency": "CNY"
    }
  ],
  "success": true
}
```
