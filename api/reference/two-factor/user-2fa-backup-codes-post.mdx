title: "重新生成备用码"
description: "POST /api/user/2fa/backup_codes"
---

接口：`POST /api/user/2fa/backup_codes`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `code` | ✓ | 6 位动态码或备用码 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/backup_codes \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```
