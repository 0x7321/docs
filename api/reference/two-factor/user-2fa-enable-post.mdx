---
title: "启用 2FA"
description: "POST /api/user/2fa/enable"
---

接口：`POST /api/user/2fa/enable`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `code` | ✓ | TOTP App 中的 6 位动态码 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/enable \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```
