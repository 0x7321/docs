接口：`GET /api/log/self`

<Warning>
**不要**加 `New-Api-User` 標頭。此路由加上該標頭會觸發不同鑑權邏輯，返回 `access token invalid`。只用 `Authorization: Bearer`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `p` | 頁碼 |
| `page_size` | 每頁數量 |
| `type` | `0` 全部、`1` 文字、`2` 圖像、`3` 音訊 |
| `channel` | 渠道 ID 篩選 |
| `model_name` | 模型名稱篩選 |
| `start_timestamp` | 開始時間（Unix 秒） |
| `end_timestamp` | 結束時間（Unix 秒） |

```bash
curl "https://bearlab.ai/api/log/self?p=1&page_size=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
