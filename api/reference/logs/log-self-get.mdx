---
title: "获取个人日志"
description: "GET /api/log/self"
---

## 请求头

| 请求头 | 必填 | 说明 |
| --- | --- | --- |
| `Authorization` | ✓ | `Bearer YOUR_ACCESS_TOKEN`，账号访问令牌（控制台登录态，非 `sk-` API Key） |

接口：`GET /api/log/self`

分页查询当前账号的调用日志。

## 查询参数

| 查询参数 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| `p` |  | integer | 页码，从 1 开始 |
| `page_size` |  | integer | 每页数量 |
| `type` |  | integer | 日志类型（`2` 消费 / `1` 充值 / `3` 管理 / `4` 系统 / `5` 错误；省略或 `0` 为全部） |
| `token_name` |  | string | 按令牌名称筛选 |
| `model_name` |  | string | 按模型名筛选 |
| `group` |  | string | 按分组筛选 |
| `start_timestamp` |  | integer | 开始时间，Unix 秒 |
| `end_timestamp` |  | integer | 结束时间，Unix 秒 |
| `request_id` |  | string | 按请求 ID 精确查询 |
| `upstream_request_id` |  | string | 按上游请求 ID 查询 |

```bash
curl "https://bearlab.ai/api/log/self?p=1&page_size=10&type=2" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `data.items` | array | 日志记录数组 |
| `type` | integer | 日志类型（同上） |
| `content` | string | 日志描述（计费倍率等） |
| `token_name` | string | 令牌名称 |
| `model_name` | string | 模型名 |
| `quota` | integer | 本次消耗额度 |
| `prompt_tokens` / `completion_tokens` | integer | 输入 / 输出 token 数 |
| `use_time` | integer | 请求耗时（秒） |
| `is_stream` | boolean | 是否流式 |
| `channel` | integer | 渠道 ID（注意 json 字段名为 `channel`，非 `channel_id`） |
| `group` | string | 使用的分组 |
| `request_id` | string | 请求 ID |
| `data.page` / `page_size` / `total` | integer | 分页信息 |

<Note>
来源：new-api 源码 `controller/log.go` `GetUserLogs` + `model/log.go` `Log`。个人日志查询**不支持** `keyword` 或 `channel` 查询参数（`/api/log/self/search` 已废弃）；`channel` 仅作为响应字段返回。
</Note>
