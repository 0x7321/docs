title: "2FA 登入"
description: "POST /api/user/login/2fa"
---

接口：`POST /api/user/login/2fa`

用於已完成第一步登入、且帳號啟用 2FA 的會話。

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `code` | ✓ | 6 位 TOTP 動態碼或備用碼 |

```bash
curl -X POST https://bearlab.ai/api/user/login/2fa \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```
