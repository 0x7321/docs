接口：`PUT /api/user/aff/code`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `aff_code` | ✓ | 新邀請碼 |

```bash
curl -X PUT https://bearlab.ai/api/user/aff/code \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"aff_code": "NEWCODE"}'
```
