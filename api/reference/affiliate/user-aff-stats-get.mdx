接口：`GET /api/user/aff/stats`

```bash
curl https://bearlab.ai/api/user/aff/stats \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "aff_balance": "0",
    "aff_code": "6ZQF",
    "aff_count": 0,
    "aff_earned": "0",
    "aff_limit_effective": 30,
    "aff_min_withdraw": 10,
    "aff_rate_effective": 5
  }
}
```

| 欄位 | 說明 |
| --- | --- |
| `aff_balance` | 當前可提現返利餘額（CNY） |
| `aff_earned` | 累計返利金額 |
| `aff_count` | 成功邀請人數 |
| `aff_rate_effective` | 有效返利比率（%） |
| `aff_min_withdraw` | 最低提現金額 |
| `aff_limit_effective` | 返利有效期（天） |
