接口：`GET /api/user/aff/withdrawals`

<Warning>
分頁參數為 `p`（**不是** `page`）。回應資料鍵為 `items`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `p` | 頁碼（注意：不是 `page`） |
| `page_size` | 每頁數量 |

```bash
curl "https://bearlab.ai/api/user/aff/withdrawals?p=1&page_size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "items": [],
    "page": 1,
    "page_size": 5,
    "total": 0
  }
}
```
