title: "两步验证"
description: "查询、初始化、启用、停用 2FA 与重新生成备用码。"
---

## 2FA 状态 `GET /api/user/2fa/status`

```bash
curl https://bearlab.ai/api/user/2fa/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{ "data": { "enabled": false, "locked": false }, "message": "", "success": true }
```

---

## 初始化 2FA `POST /api/user/2fa/setup`

产生 TOTP secret 与 QR Code 数据。调用后仍需用 `POST /api/user/2fa/enable` 验证动态码才会启用。

```bash
curl -X POST https://bearlab.ai/api/user/2fa/setup \
  -H "Authorization: Bearer YOUR_API_KEY"
```

---

## 启用 2FA `POST /api/user/2fa/enable`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `code` | ✓ | TOTP App 中的 6 位动态码 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/enable \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

---

## 停用 2FA `POST /api/user/2fa/disable`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `code` | ✓ | 6 位动态码或备用码 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/disable \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

---

| 请求体字段 | 必填 | 说明 |

```bash

  -H "Authorization: Bearer YOUR_API_KEY" \
```
