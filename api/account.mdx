所有端點均需 `Authorization: Bearer YOUR_API_KEY` 標頭，基底 URL 為 `https://bearlab.ai`。

---

## 取得當前使用者 `GET /api/user/self`

返回登入帳號的完整資訊（50+ 欄位）。

<Warning>
`user.id` 不存在於此回應中。`user_id` 僅在 `/api/user/checkin-history` 和 `PUT /api/token/` 回應中可見。回應中含有 `access_token` 明文欄位（即當前 Bearer token 本身）及 `DeletedAt` 欄位（GORM 軟刪除欄位，正常情況下為 `null`）。
</Warning>

```bash
curl https://bearlab.ai/api/user/self \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 更新使用者資訊 `PUT /api/user/self`

更新顯示名稱、語言、頭像等基本設定。

```bash
curl -X PUT https://bearlab.ai/api/user/self \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"display_name":"My Name","language":"zh-Hant"}'
```

## 取得存取令牌 `GET /api/user/token`

<Warning>
**破壞性操作**：每次呼叫都會重新生成 `access_token`，舊令牌立即失效。此令牌用於系統層面鑑權，與 API Key (`/api/token/`) 不同。
</Warning>

```bash
curl https://bearlab.ai/api/user/token \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": "DmGxCrprnCE7dHTFUGK2dCZ/3T1U+hA=",
  "success": true
}
```

---

## 登出 `GET /api/user/logout`

使當前 session 失效。

```bash
curl https://bearlab.ai/api/user/logout \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{ "data": null, "message": "", "success": true }
```

## KYC 認證狀態 `GET /api/user/verify/self`

```json 回應
{
  "data": {
    "user_id": 3930,
    "type": "",
    "status": "",
    "created_at": "2026-06-21T06:47:15.368+08:00",
    "updated_at": "2026-06-21T06:47:15.368+08:00"
  }
}
```

| `type` 值 | 說明 |
| --- | --- |
| `""` | 未認證 |
| `"personal"` | 個人認證 |
| `"enterprise"` | 企業認證 |
| `"passport"` | 護照認證 |

| `status` 值 | 說明 |
| --- | --- |
| `""` | 未提交 |
| `"PROCESSING"` | 審核中 |
| `"SUCCESS"` | 已通過 |
| `"FAIL"` | 已拒絕 |

## 2FA 狀態 `GET /api/user/2fa/status`

返回當前使用者的兩步驗證狀態。

```bash
curl https://bearlab.ai/api/user/2fa/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{ "data": { "enabled": false, "locked": false }, "message": "", "success": true }
```

---

## 初始化 2FA `POST /api/user/2fa/setup`

產生 TOTP secret 與 QR Code 資料。呼叫後仍需用 `POST /api/user/2fa/enable` 驗證動態碼才會啟用。

```bash
curl -X POST https://bearlab.ai/api/user/2fa/setup \
  -H "Authorization: Bearer YOUR_API_KEY"
```

---

## 啟用 2FA `POST /api/user/2fa/enable`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `code` | ✓ | TOTP App 中的 6 位動態碼 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/enable \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

---

## 停用 2FA `POST /api/user/2fa/disable`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `code` | ✓ | 6 位動態碼或備用碼 |

```bash
curl -X POST https://bearlab.ai/api/user/2fa/disable \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

---

## 刪除 Passkey `DELETE /api/user/passkey`

```bash
curl -X DELETE https://bearlab.ai/api/user/passkey \
  -H "Authorization: Bearer YOUR_API_KEY"
```

---

## 開始註冊 Passkey `POST /api/user/passkey/register/begin`

```bash
curl -X POST https://bearlab.ai/api/user/passkey/register/begin \
  -H "Authorization: Bearer YOUR_API_KEY"
```

---

## 完成註冊 Passkey `POST /api/user/passkey/register/finish`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `credential` | ✓ | WebAuthn 註冊憑證回應物件 |

```bash
curl -X POST https://bearlab.ai/api/user/passkey/register/finish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"credential":{}}'
```

---

## 開始驗證 Passkey `POST /api/user/passkey/verify/begin`

```bash
curl -X POST https://bearlab.ai/api/user/passkey/verify/begin \
  -H "Authorization: Bearer YOUR_API_KEY"
```

---

## 完成驗證 Passkey `POST /api/user/passkey/verify/finish`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `credential` | ✓ | WebAuthn 驗證憑證回應物件 |

```bash
curl -X POST https://bearlab.ai/api/user/passkey/verify/finish \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"credential":{}}'
```

<Note>
本實例目前回應 Passkey 登入未啟用，表示路由存在但平台設定未開啟。
</Note>

## 簽到記錄 `GET /api/user/checkin-history`

<Note>
`user_id` 在此端點可見，而 `/api/user/self` 不返回 `id` 欄位。
</Note>

```json 回應
{
  "data": [
    {
      "id": 15987,
      "user_id": 3930,
      "check_in_at": "2026-06-20T00:00:00+08:00",
      "quota": "0.0128"
    }
  ]
}
```
