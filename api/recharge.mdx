## 付款方式 `GET /api/user/payment`

列出当前用户可用的付款方式。

```bash
curl https://bearlab.ai/api/user/payment \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": [
    {
      "type": "epay",
      "uuid": "98de8f0bc169...",
      "name": "微信",
      "icon": "wxpay",
      "min_amount": "10",
      "max_amount": "10000",
      "currency": "CNY"
    }
  ],
  "success": true
}
```

## 订单历史 `GET /api/payment/order/self`

| 参数 | 说明 |
| --- | --- |
| `page` | 页码 |
| `page_size` | 每页数量 |

```bash
curl "https://bearlab.ai/api/payment/order/self?page=1&page_size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

| `status` | 说明 |
| --- | --- |
| `pending` | 待支付 |
| `paid` | 已支付 |
| `closed` | 已关闭（已付款订单完成后） |
| `cancelled` | 已取消 |
| `refunded` | 已退款 |

## 创建订单 `POST /api/user/order`

| 请求体字段 | 必填 | 说明 |
| --- | --- | --- |
| `uuid` | ✓ | 付款方式 UUID，来自 `/api/user/payment` |
| `amount` | ✓ | 充值金额 |
| `promo_code` |  | 优惠码，可省略或空字符串 |

```bash
curl -X POST https://bearlab.ai/api/user/order \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"uuid":"98de8f0bc169...","amount":10,"promo_code":"PROMO123"}'
```

---

## 查询订单状态 `GET /api/user/order/status`

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| `trade_no` | ✓ | 订单号，来自创建订单响应 |

```bash
curl "https://bearlab.ai/api/user/order/status?trade_no=USR3930NOxxx" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
