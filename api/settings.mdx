## 推播設定 `GET /api/push/self`

<Warning>
`/api/user/setting` 端點在 6.3.13 中不存在（GET 和 PUT 均返回 404）。設定讀取走 `/api/user/self` 或 `/api/push/self`，設定更新走 `POST /api/push/self`。
</Warning>

<Note>
`subscription_options` 是逗號分隔字串，不是陣列。`notice_email` / `webhook_url` / `webhook_token` 雖然在原始碼中存在，但實際回應中不包含這些欄位。
</Note>

```bash
curl https://bearlab.ai/api/push/self \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "user_id": 3930,
    "subscription_options": "quota_push,sale_push,back_push,system_push,price_push,ticket_event",
    "notice_type": "email",
    "tip_quota": "1"
  },
  "message": "",
  "success": true
}
```

| `subscription_options` 項目 | 說明 |
| --- | --- |
| `quota_push` | 餘額提醒 |
| `sale_push` | 促銷活動 |
| `back_push` | 服務恢復通知 |
| `system_push` | 系統公告 |
| `price_push` | 價格變動 |
| `ticket_event` | 工單更新 |

## 更新推播設定 `POST /api/push/self`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `notice_type` | ✓ | 通知方式，如 `email` |
| `notice_email` |  | 通知郵箱 |
| `webhook_url` |  | Webhook 地址 |
| `webhook_token` |  | Webhook token |
| `tip_quota` |  | 餘額提醒閾值 |

```bash
curl -X POST https://bearlab.ai/api/push/self \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "notice_type": "email",
    "notice_email": "user@example.com",
    "webhook_url": "",
    "webhook_token": "",
    "tip_quota": "10"
  }'
```

## 自動充值設定 `GET /api/auto-recharge/settings`

<Warning>
路徑使用連字符 `auto-recharge`。下劃線版本 `/api/auto_recharge/self` 返回 404。
</Warning>

```bash
curl https://bearlab.ai/api/auto-recharge/settings \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "enabled": false,
    "status": "not_configured"
  },
  "message": "",
  "success": true
}
```

完整啟用後的欄位：`enabled` / `status` / `threshold_amount` / `target_amount` / `gateway_uuid` / `payment_method_label`。

---

## 更新自動充值 `PUT /api/auto-recharge/settings`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `enabled` | ✓ | 是否啟用自動充值 |
| `threshold_amount` | 啟用時必填 | 觸發充值的餘額閾值 |
| `target_amount` | 啟用時必填 | 充值後目標餘額 |
| `gateway_uuid` | 啟用時必填 | 付款方式 UUID，來自 `/api/user/payment` |

```bash
curl -X PUT https://bearlab.ai/api/auto-recharge/settings \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "enabled": true,
    "threshold_amount": 10,
    "target_amount": 50,
    "gateway_uuid": "98de8f0bc169..."
  }'
```

## 不支援或已替代的使用者端點

| 路徑 | 實測結果 / 替代方式 |
| --- | --- |
| `GET/PUT /api/user/setting` | 404；使用 `/api/push/self` |
| `GET/PUT /api/user/security` | 404 |
| `GET /api/user/groups` | 404 |
| `GET /api/user/self/groups` | 404 |
| `GET /api/2fa/status` | 404；正確路徑是 `/api/user/2fa/status` |
| `GET /api/auto_recharge/self` | 404；正確路徑是 `/api/auto-recharge/settings` |
