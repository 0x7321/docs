## 付款方式 `GET /api/user/payment`

列出平台支援的所有付款方式。

```bash
curl https://bearlab.ai/api/user/payment \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": [
    {
      "id": 0,
      "type": "epay",
      "uuid": "98de8f0bc169...",
      "name": "微信",
      "icon": "wxpay",
      "min_amount": "10",
      "max_amount": "10000",
      "currency": "CNY",
      "fixed_fee": 0,
      "percent_fee": 0,
      "sort": 1,
      "require_verification": false,
      "display_language": "all",
      "enable_invoice": false,
      "created_at": 0
    }
  ]
}
```

## 訂單歷史 `GET /api/payment/order/self`

<Warning>
路徑為 `/api/payment/order/self`，**不是** `/api/order/*` 或 `/api/topup/record`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `page` | 頁碼 |
| `page_size` | 每頁數量 |

```bash
curl "https://bearlab.ai/api/payment/order/self?page=1&page_size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "items": [
      {
        "id": 6701,
        "gateway_id": 2,
        "trade_no": "USR3930NO9E0nhH1781976862",
        "gateway_no": "",
        "amount": "11",
        "order_amount": "11",
        "order_currency": "CNY",
        "quota": "11",
        "fee": 0,
        "discount": 0,
        "status": "pending",
        "invoice_status": "unavailable",
        "promo_code_id": 0,
        "usage_id": 0,
        "refund_amount": "0",
        "created_at": 1781976863
      }
    ],
    "page": 1,
    "page_size": 5,
    "total": 2
  }
}
```

| `status` | 說明 |
| --- | --- |
| `pending` | 待支付 |
| `paid` | 已支付 |
| `closed` | 已關閉（已付款訂單完成後） |
| `cancelled` | 已取消 |
| `refunded` | 已退款 |

## 建立訂單 `POST /api/user/order`

建立充值訂單，返回支付跳轉連結。

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `uuid` | ✓ | 付款方式 UUID，來自 `/api/user/payment` |
| `amount` | ✓ | 充值金額 |
| `promo_code` |  | 優惠碼，可省略或空字串 |

```bash
curl -X POST https://bearlab.ai/api/user/order \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "uuid": "98de8f0bc169...",
    "amount": 10,
    "promo_code": "PROMO123"
  }'
```

```json 回應
{
  "data": {
    "trade_no": "USR3930NOxxx",
    "pay_url": "https://..."
  },
  "success": true
}
```

## 查詢訂單狀態 `GET /api/user/order/status`

| 參數 | 必填 | 說明 |
| --- | --- | --- |
| `trade_no` | ✓ | 訂單號，來自建立訂單的回應 |

```bash
curl "https://bearlab.ai/api/user/order/status?trade_no=USR3930NOxxx" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 兌換碼充值 `POST /api/user/topup`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `key` | ✓ | 兌換碼 / 充值碼 |

```bash
curl -X POST https://bearlab.ai/api/user/topup \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"key": "TOPUP-CODE-XXXX"}'
```

## 驗證優惠碼 `POST /api/user/promo/validate`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `code` | ✓ | 要驗證的優惠碼 |

<Warning>
回應欄位為 `message` + `valid`（**不是** `valid` + `discount` + `type`）。無效優惠碼時 `success` 仍為 `true`，須檢查 `valid` 欄位。
</Warning>

```bash
curl -X POST https://bearlab.ai/api/user/promo/validate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"code": "PROMO123"}'
```

```json 回應（無效碼）
{
  "data": { "message": "Invalid promo code", "valid": false },
  "success": true
}
```

```json 回應（有效碼）
{
  "data": { "message": "", "valid": true },
  "success": true
}
```
