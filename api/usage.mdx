## 用量統計 `GET /api/data/self`

<Warning>
**RixAPI 專屬格式**：回應頂層無 `data` 包裝，`statistics` / `time_series` / `model_distribution` 直接在根物件。舊版 new-api 返回 `{ "data": [] }` 扁平陣列，格式完全不同。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `start_timestamp` | 開始時間（Unix 秒） |
| `end_timestamp` | 結束時間（Unix 秒） |
| `granularity` | `day` 或 `hour` |
| `dimension` | `model` 或其他維度 |

```bash
curl "https://bearlab.ai/api/data/self?granularity=day" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "statistics": {

    "success_rate": 0,

    "avg_tokens": 0,
    "avg_rpm": 0,
    "avg_tpm": 0,
    "total_input_tokens": 0,
    "total_cache_tokens": 0,
    "total_cache_creation": 0,
    "total_cache_creation_5m": 0,
    "total_cache_creation_1h": 0,
```

## 單模型可用率 `GET /api/data/uptime`

<Warning>
`model_name` 為**必填**參數。不帶此參數返回 `{ "success": false, "message": "模型名稱不能為空" }`。
</Warning>

| 參數 | 必填 | 說明 |
| --- | --- | --- |
| `model_name` | ✓ | 模型 ID，如 `claude-opus-4-6` |

```bash
curl "https://bearlab.ai/api/data/uptime?model_name=claude-opus-4-6" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應（推斷結構）
{
  "data": {
    "points": [
      {
        "timestamp": 1781967600000,
        "token_group": "default",
        "success_rate": 98,
        "avg_latency": 1200,
        "token_speed": 45.0
      }
    ],
    "group_summary": {},
    "percentile_history": [
      {
        "timestamp": 1781967600000,
        "latency_min": 800,
        "latency_p50": 1200,
        "latency_p90": 2000,
        "latency_p99": 3500
      }
    ]
  }
}
```

## 批次可用率 `POST /api/data/uptime/batch`

同時查詢多個模型的可用率數據，每個模型按分組細分。

```bash
curl -X POST https://bearlab.ai/api/data/uptime/batch \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model_names": ["claude-opus-4-6", "claude-sonnet-4-6"], "token_group": "default"}'
```

```json 回應
{
  "data": {
    "claude-opus-4-6": {
      "points": [
        {
          "timestamp": 1781967600000,
          "token_group": "default",
          "success_rate": 100,
          "avg_latency": 4264,
          "token_speed": 31.37
        }
      ],
      "group_summary": {
        "default": { "avg_latency": 4264, "token_speed": 31.37 }
      },
      "group_percentiles": {
        "claude-code-special": {
          "latency": { "p50": 2734, "p90": 3890, "p99": 4150, "min": 800 }
        }
      }
    }
  }
}
```

| 欄位 | 說明 |
| --- | --- |
| `points` | 時序數據點列表 |
| `group_summary` | 各分組的彙總延遲與速度（只含有數據的分組） |
| `group_percentiles` | 各 token_group 的延遲百分位數（p50/p90/p99/min）；key 為 group 名稱，非模型名稱 |

<Note>
`group_percentiles` 的 key 是 **token_group 名稱**（如 `claude-code-special`），不是請求的 `model_names`。它統計的是跨所有模型、該 group 的全局延遲分布。
</Note>
