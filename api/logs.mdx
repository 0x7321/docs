## 呼叫記錄 `GET /api/log/self`

<Warning>
**不要**加 `New-Api-User` 標頭。此路由加上該標頭會觸發不同鑑權邏輯，返回 `access token invalid`。只用 `Authorization: Bearer`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `p` | 頁碼 |
| `page_size` | 每頁數量 |
| `type` | `0` 全部、`1` 文字、`2` 圖像、`3` 音訊 |
| `channel` | 渠道 ID 篩選 |
| `model_name` | 模型名稱篩選 |
| `start_timestamp` | 開始時間（Unix 秒） |
| `end_timestamp` | 結束時間（Unix 秒） |

```bash
curl "https://bearlab.ai/api/log/self?p=1&page_size=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "items": [
      {
        "id": 123,
        "created_at": 1781977779,
        "type": 1,
        "content": "...",
        "username": "linuxdo_3930",
        "model_name": "claude-opus-4-6",
        "quota": "0.001",
        "prompt_tokens": 100,
        "completion_tokens": 50,
        "channel": 5,
        "request_id": "req_xxx"
      }
    ],
    "page": 1,
    "page_size": 20,
    "total": 0
  }
}
```

## 即時統計 `GET /api/log/self/stat`

返回指定時間範圍內的用量彙總。

| 參數 | 說明 |
| --- | --- |
| `start_timestamp` | 開始時間（Unix 秒） |
| `end_timestamp` | 結束時間（Unix 秒） |
| `type` | 同上 |

```bash
curl "https://bearlab.ai/api/log/self/stat?start_timestamp=1781900000&end_timestamp=1782000000" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "quota": 0,
    "rpm": 0,
    "tpm": 0,
    "mpm": 0,
    "unfinished_count": 0,
    "unfinished_by_vendor": {}
  }
}
```

| 欄位 | 說明 |
| --- | --- |
| `quota` | 消耗額度 |
| `rpm` | 每分鐘請求數 |
| `tpm` | 每分鐘 Token 數 |
| `mpm` | 每分鐘 Message 數 |
| `unfinished_count` | 未完成請求數 |
