## 我的邀請碼 `GET /api/user/aff`

<Warning>
此端點**只返回邀請碼字串**，不含統計數據。完整資訊請用 `GET /api/user/aff/stats`。
</Warning>

```bash
curl https://bearlab.ai/api/user/aff \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{ "data": "6ZQF" }
```

## 返利統計 `GET /api/user/aff/stats`

```bash
curl https://bearlab.ai/api/user/aff/stats \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "aff_balance": "0",
    "aff_code": "6ZQF",
    "aff_count": 0,
    "aff_earned": "0",
    "aff_limit_effective": 30,
    "aff_min_withdraw": 10,
    "aff_rate_effective": 5
  }
}
```

| 欄位 | 說明 |
| --- | --- |
| `aff_balance` | 當前可提現返利餘額（CNY） |
| `aff_earned` | 累計返利金額 |
| `aff_count` | 成功邀請人數 |
| `aff_rate_effective` | 有效返利比率（%） |
| `aff_min_withdraw` | 最低提現金額 |
| `aff_limit_effective` | 返利有效期（天） |

## 返利記錄 `GET /api/user/aff/logs`

<Warning>
回應資料鍵為 `logs`，**不是** `items`。分頁使用 `page` / `page_size` / `total`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `page` | 頁碼 |
| `page_size` | 每頁數量 |

```bash
curl "https://bearlab.ai/api/user/aff/logs?page=1&page_size=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "logs": [],
    "page": 1,
    "page_size": 5,
    "total": 0
  }
}
```

## 提現記錄 `GET /api/user/aff/withdrawals`

<Warning>
分頁參數為 `p`（**不是** `page`）。回應資料鍵為 `items`。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `p` | 頁碼（注意：不是 `page`） |
| `page_size` | 每頁數量 |

```bash
curl "https://bearlab.ai/api/user/aff/withdrawals?p=1&page_size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "items": [],
    "page": 1,
    "page_size": 5,
    "total": 0
  }
}
```

## 修改邀請碼 `PUT /api/user/aff/code`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `aff_code` | ✓ | 新邀請碼 |

```bash
curl -X PUT https://bearlab.ai/api/user/aff/code \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"aff_code": "NEWCODE"}'
```

## 返利轉餘額 `POST /api/user/aff/convert`

<Warning>
**破壞性操作**：將 `aff_balance`（返利餘額）轉入帳號主餘額，不可逆。
</Warning>

```bash
curl -X POST https://bearlab.ai/api/user/aff/convert \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 申請提現 `POST /api/user/aff/withdraw`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `amount` | ✓ | 提現金額 |
| `payment_info` | ✓ | 收款資訊物件；如 `{ "alipay": "user@example.com" }` |

```bash
curl -X POST https://bearlab.ai/api/user/aff/withdraw \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 10,
    "payment_info": {
      "alipay": "user@example.com"
    }
  }'
```
