本頁只列使用者身份驗證流程。已登入後的帳號資料、2FA 設定與 Passkey 綁定管理見「帳號管理」。

<Warning>
登入、註冊、驗證碼與重設密碼端點可能受 CAPTCHA 保護。實測未通過 CAPTCHA 時會返回 `need_captcha: true`，不是路由不存在。
</Warning>

---

## 使用者登入 `POST /api/user/login`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `username` | ✓ | 使用者名稱或登入帳號 |
| `password` | ✓ | 密碼 |

```bash
curl -X POST https://bearlab.ai/api/user/login \
  -H "Content-Type: application/json" \
  -d '{"username":"YOUR_USERNAME","password":"YOUR_PASSWORD"}'
```

```json 回應（需要 CAPTCHA 時）
{
  "data": { "need_captcha": true },
  "message": "Please complete the verification",
  "success": false
}
```

---

## 2FA 登入 `POST /api/user/login/2fa`

用於已完成第一步登入、且帳號啟用 2FA 的會話。

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `code` | ✓ | 6 位 TOTP 動態碼或備用碼 |

```bash
curl -X POST https://bearlab.ai/api/user/login/2fa \
  -H "Content-Type: application/json" \
  -d '{"code":"123456"}'
```

## 登出 `GET /api/user/logout`

```bash
curl https://bearlab.ai/api/user/logout \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 使用者註冊 `POST /api/user/register`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `username` | ✓ | 使用者名稱 |
| `password` | ✓ | 密碼 |
| `email` | ✓ | 郵箱 |
| `verification_code` | ✓ | 郵箱驗證碼 |
| `aff_code` |  | 邀請碼，可空字串 |

```bash
curl -X POST https://bearlab.ai/api/user/register \
  -H "Content-Type: application/json" \
  -d '{"username":"YOUR_USERNAME","password":"YOUR_PASSWORD","email":"user@example.com","verification_code":"123456","aff_code":""}'
```

```json 回應（需要 CAPTCHA 時）
{
  "data": { "need_captcha": true },
  "message": "Please complete the verification",
  "success": false
}
```

## 發送郵箱驗證碼 `GET /api/verification`

| 參數 | 說明 |
| --- | --- |
| `email` | 接收驗證碼的郵箱 |

```bash
curl "https://bearlab.ai/api/verification?email=user@example.com"
```

---

## 發送密碼重設郵件 `GET /api/reset_password`

| 參數 | 說明 |
| --- | --- |
| `email` | 需要重設密碼的郵箱 |

```bash
curl "https://bearlab.ai/api/reset_password?email=user@example.com"
```

---

## 重設密碼 `POST /api/user/reset`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `email` | ✓ | 帳號郵箱 |
| `token` | ✓ | 重設密碼郵件中的 token |
| `password` | ✓ | 新密碼 |

```bash
curl -X POST https://bearlab.ai/api/user/reset \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","token":"RESET_TOKEN","password":"NEW_PASSWORD"}'
```

---

## 開始 Passkey 登入 `POST /api/user/passkey/login/begin`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `username` | ✓ | 使用者名稱 |

```bash
curl -X POST https://bearlab.ai/api/user/passkey/login/begin \
  -H "Content-Type: application/json" \
  -d '{"username":"YOUR_USERNAME"}'
```

---

## 完成 Passkey 登入 `POST /api/user/passkey/login/finish`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `credential` | ✓ | WebAuthn 登入憑證回應物件 |

```bash
curl -X POST https://bearlab.ai/api/user/passkey/login/finish \
  -H "Content-Type: application/json" \
  -d '{"credential":{}}'
```

<Note>
本實例目前回應 Passkey 登入未啟用，表示路由存在但平台設定未開啟。
</Note>
