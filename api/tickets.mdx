<Warning>
工單分頁欄位與其他端點**不同**：`size`（非 `page_size`）、`total_count`（非 `total`）、`data`（非 `items`）。
</Warning>

## 工單列表 `GET /api/ticket/`

<Warning>
路徑必須帶**尾部斜杠** `/api/ticket/`。不帶斜杠時伺服器返回 **301 重定向**，curl 需加 `-L` 跟隨，否則拿不到資料。
</Warning>

| 參數 | 說明 |
| --- | --- |
| `page` | 頁碼 |
| `size` | 每頁數量（注意：不是 `page_size`） |

```bash
curl -L "https://bearlab.ai/api/ticket/?page=1&size=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "data": [],
    "page": 1,
    "size": 10,
    "total_count": 0
  }
}
```

<Note>
外層的 `data` 是標準 API 包裝，內層的 `data` 是工單陣列。這是雙層 `data` 結構。實測確認 `size` 欄位由查詢參數決定（預設 10）。
</Note>

## 工單統計 `GET /api/ticket/stats`

```bash
curl https://bearlab.ai/api/ticket/stats \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應
{
  "data": {
    "by_priority": {},
    "by_status": {},
    "total": 0
  }
}
```

## 工單詳情 `GET /api/ticket/{id}`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

```bash
curl https://bearlab.ai/api/ticket/123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json 回應（推斷）
{
  "data": {
    "id": 123,
    "subject": "問題標題",
    "status": "open",
    "content": "問題描述",
    "replies": [
      {
        "id": 1,
        "content": "回覆內容",
        "created_at": 1781977779
      }
    ]
  }
}
```

## 建立工單 `POST /api/ticket`

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `subject` | ✓ | 工單標題 |
| `content` | ✓ | 工單內容 |

```bash
curl -X POST https://bearlab.ai/api/ticket \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "問題標題",
    "content": "請描述您遇到的問題..."
  }'
```

## 回覆工單 `POST /api/ticket/{id}/reply`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `content` | ✓ | 回覆內容 |

```bash
curl -X POST https://bearlab.ai/api/ticket/123/reply \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "補充說明..."}'
```

## 更新工單狀態 `PUT /api/ticket/{id}/status`

| 路徑參數 | 說明 |
| --- | --- |
| `id` | 工單 ID |

| 請求體欄位 | 必填 | 說明 |
| --- | --- | --- |
| `status` | ✓ | 目標狀態 |

```bash
curl -X PUT https://bearlab.ai/api/ticket/123/status \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"status": "closed"}'
```

| `status` 值 | 說明 |
| --- | --- |
| `open` | 開啟中 |
| `in_progress` | 處理中 |
| `closed` | 已關閉 |
