## 服务区域

BearLab AI（bearlab.ai）面向**美国与欧盟**司法管辖区直接提供服务。

### 亚太区域

亚太区域由**印度区域合作伙伴**提供服务。

该区域的 API 请求须改用专用域名 `https://bearlab.in`，而非 `https://bearlab.ai`。

该区域服务的相关政策由区域合作伙伴自行制定；亚太区域用户请自行确保符合所在地的法律法规。

| 区域 | API 域名 | OpenAI | Anthropic |
| --- | --- | --- | --- |
| 美国 / 欧盟 | `https://bearlab.ai` | `https://bearlab.ai/v1/chat/completions` | `https://bearlab.ai/v1/messages` |
| 亚太区域 | `https://bearlab.in` | `https://bearlab.in/v1/chat/completions` | `https://bearlab.in/v1/messages` |
