对话补全是核心端点，通过三种兼容接口（OpenAI、Anthropic、Gemini）提供，共享一个密钥。

<Note>
  BearLab AI 服务两个地区。美国/欧洲用户调用 `https://bearlab.ai`；亚太地区用户调用专用域名 `https://bearlab.space`。两个地区共享相同的密钥和模型目录。见 [服务可用性](/zh-my/guides/availability)。
</Note>

## 端点

| 协议 | 方法与路径（美国/欧洲） | 方法与路径（亚太地区） |
| --- | --- | --- |
| OpenAI 兼容 | `POST https://bearlab.ai/v1/chat/completions` | `POST https://bearlab.space/v1/chat/completions` |
| Anthropic 兼容 | `POST https://bearlab.ai/v1/messages` | `POST https://bearlab.space/v1/messages` |
| Gemini 原生 | `base_url=https://bearlab.ai`, `api_version=v1beta`（Google genai SDK） | `base_url=https://bearlab.space`, `api_version=v1beta` |

使用 API 密钥（`sk-...`）在 `Authorization: Bearer` 标头中验证每个请求。Anthropic 协议另外支持 `x-api-key` 标头。

## 常见参数

| 参数 | 必需 | 说明 |
| --- | --- | --- |
| `model` | 是 | 模型 ID。见 [模型与定价 API](/zh-my/guides/models)。 |
| `messages` | 是 | 有序的对话消息数组。 |
| `max_tokens` | OpenAI：可选 · Anthropic：**必需** | 最大输出 token 数。 |
| `temperature` | 否 | 采样温度。OpenAI 范围 `0`-`2`；Anthropic 范围 `0`-`1`。 |
| `stream` | 否 | 当为 `true` 时通过 SSE 流式传输响应。 |
| `tools` | 否 | 工具调用的函数/工具定义。 |

<Warning>
  协议差异：OpenAI 协议将系统提示词放在 `messages` 中作为 `role: system`，`max_tokens` 为可选。Anthropic 协议使用顶层 `system` 字段且**必需** `max_tokens`。
</Warning>

## 流式传输

设置 `stream: true` 通过 Server-Sent Events 接收响应。

- **OpenAI 协议** — 增量 `delta` 数据块，以最终 `data: [DONE]` 行结束。
- **Anthropic 协议** — 类型化 SSE 事件（`message_start`、`content_block_delta`、`message_stop` 等）；没有 `[DONE]` 哨兵。

<CodeGroup>
```python 美国/欧洲
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.ai/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```

```python 亚太地区
from openai import OpenAI

client = OpenAI(api_key="sk-...", base_url="https://bearlab.space/v1")

stream = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Write a short poem about spring."}],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)
```
</CodeGroup>

## 工具调用

OpenAI 和 Anthropic 协议都支持工具调用。

```python
from openai import OpenAI

client = OpenAI(base_url="https://bearlab.ai/v1", api_key="sk-...")

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the current weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"],
        },
    },
}]

resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "What's the weather in London?"}],
    tools=tools,
)
print(resp.choices[0].message)
```

<Note>
  亚太地区用户：改为设置 `base_url="https://bearlab.space/v1"`。
</Note>

## Gemini 原生协议

Gemini 模型可以用原生 Google AI Studio 格式调用，无需转换为 OpenAI 格式。

<CodeGroup>
```python 美国/欧洲
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.ai"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```

```python 亚太地区
from google import genai
from google.genai import types

client = genai.Client(
    api_key="sk-...",
    http_options=types.HttpOptions(api_version="v1beta", base_url="https://bearlab.space"),
)

resp = client.models.generate_content(model="gemini-3.1-pro-preview", contents="Hello")
print(resp.text)
```
</CodeGroup>

<Warning>
  1. 将 `base_url` 设置为地区根目录（`https://bearlab.ai` 或 `https://bearlab.space`）；不要追加 `/v1beta`，SDK 会自动添加。
  2. 将 `api_version` 设置为 `v1beta`。
  3. 安装：Python `pip install google-genai`，Node.js `npm install @google/genai`。
</Warning>

<Tip>
  重复的长前缀（固定系统提示词、长文档）可以使用 [提示词缓存](/zh-my/guides/prompt-caching) 降低重复部分的成本。
</Tip>
