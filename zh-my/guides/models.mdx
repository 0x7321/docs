本页列出您的 API 密钥可以调用的模型，按系列分组。在聊天请求的 `model` 字段中传入模型的确切 `id`。

<Note>
  亚太地区用户：请改用专用域名 `https://bearlab.space` 而非 `https://bearlab.ai`。详见 [服务可用性](/zh-my/guides/availability)。
</Note>

## 如何阅读本目录

您看到的目录受限于 **您的令牌群组**。每个 API 密钥属于一个模型群组，模型只有在您的群组被允许调用时才会出现在您的清单中。下方的模型是当前的正式生产阵容；您的密钥实际看到的子集取决于您的群组。

| 概念 | 含义 |
| --- | --- |
| **模型 `id`** | 您在 `model` 字段中传送的确切字符串。区分大小写，请逐字复制。 |
| **模型群组** | 您的密钥所属的访问层级。决定您可以调用哪些模型以及以什么价格。 |
| **端点类型** | 模型接受的协议：OpenAI 兼容、Anthropic 或 Gemini。 |

## 获取即时清单

`/v1/models` 会返回您的密钥可以调用的模型，采用 OpenAI SDK 格式。它需要您的 API 密钥。回应格式遵循您的请求标头所暗示的格式（默认为 OpenAI；使用 `x-api-key` + `anthropic-version` 时为 Anthropic；使用 `x-goog-api-key` 时为 Gemini）。

<Tabs>
  <Tab title="美国 / 欧盟 (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
  <Tab title="亚太地区 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/models \
      -H "Authorization: Bearer sk-..."
    ```
  </Tab>
</Tabs>

回应（截断）：

```json
{
  "object": "list",
  "data": [
    {
      "id": "claude-opus-4-8",
      "object": "model",
      "created": 1626777600,
      "owned_by": "openai",
      "supported_endpoint_types": ["openai"]
    }
  ]
}
```

| 字段 | 描述 |
| --- | --- |
| `id` | 模型标识符；在聊天请求的 `model` 字段中传入此值 |
| `object` | 始终为 `model` |
| `created` | 发布时间（Unix 秒） |
| `owned_by` | 名义所有者字符串 |
| `supported_endpoint_types` | 接受此模型的端点：`openai`、`anthropic` 和/或 `gemini` |

## 调用一个模型

在 `model` 字段中传送确切的 `id`。两个示例域名均已显示；请选择适合您地区的那个。

<Tabs>
  <Tab title="美国 / 欧盟 (bearlab.ai)">
    ```bash
    curl https://bearlab.ai/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
  <Tab title="亚太地区 (bearlab.space)">
    ```bash
    curl https://bearlab.space/v1/chat/completions \
      -H "Authorization: Bearer sk-..." \
      -H "Content-Type: application/json" \
      -d '{
        "model": "claude-sonnet-4-6",
        "messages": [{"role": "user", "content": "Introduce yourself in one sentence"}]
      }'
    ```
  </Tab>
</Tabs>
