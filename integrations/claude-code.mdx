[Claude Code](https://docs.anthropic.com/zh-TW/docs/claude-code/overview) 是 Anthropic 官方 AI 程式設計 CLI，原生走 Anthropic 協議接入。

## 支援的模型

| 模型 | ID | 適用場景 |
| --- | --- | --- |
| Claude Opus 4.8 | `claude-opus-4-8` | 最強推理，複雜架構與深度任務 |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | 均衡效能，日常開發首選 |
| Claude Haiku 4.5 | `claude-haiku-4-5` | 最快，輕量補全與快速問答 |

## 配置

透過環境變數指向平臺端點：

```bash
export ANTHROPIC_BASE_URL="https://bearlab.ai"
export ANTHROPIC_API_KEY="YOUR_API_KEY"

claude
```

`ANTHROPIC_API_KEY` 與 `ANTHROPIC_AUTH_TOKEN` 均被識別，效果完全相同，任選其一即可。

## 指定模型（可選）

```bash
export ANTHROPIC_MODEL="claude-sonnet-4-6"           # 覆蓋預設主力模型
export ANTHROPIC_SMALL_FAST_MODEL="claude-haiku-4-5" # 覆蓋內部快速小任務用模型
```

不設定時，Claude Code 使用自身預設的模型選擇邏輯。

## 持久化

```bash
echo 'export ANTHROPIC_BASE_URL="https://bearlab.ai"' >> ~/.zshrc
echo 'export ANTHROPIC_API_KEY="YOUR_API_KEY"' >> ~/.zshrc
```

<Note>
  `ANTHROPIC_BASE_URL` 填平臺根域，不加 `/v1`。模型 ID 必須是 [模型介面](/guides/models) 中真實存在的 ID。
</Note>
