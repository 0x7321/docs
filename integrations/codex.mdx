[Codex CLI](https://github.com/openai/codex) 是 OpenAI 官方命令列程式設計代理，透過自定義 provider 接入平臺。

## 配置

編輯 `~/.codex/config.toml`，新增一個指向平臺的 provider：

```toml
# 預設模型與 provider
model = "gpt-4o"
model_provider = "bearlab"

# 第三方 provider 必須關閉響應儲存
disable_response_storage = true

[model_providers.bearlab]
name = "BearLab AI"
base_url = "https://bearlab.ai/v1"
env_key = "BEARLAB_API_KEY"
wire_api = "responses"
```

然後匯出金鑰並執行：

```bash
export BEARLAB_API_KEY="YOUR_API_KEY"
codex
```

<Warning>
  `base_url` 用 OpenAI 相容地址（`/v1`）。第三方 provider 必須設 `wire_api = "responses"`（Codex 唯一支援的協議）與 `disable_response_storage = true`，否則部分功能會報錯。`model` 必須是 [模型介面](/guides/models) 中存在的 ID。
</Warning>
