[Codex CLI](https://github.com/openai/codex) 是 OpenAI 官方命令行编程代理，通过自定义 provider 接入平台。

## 配置

编辑 `~/.codex/config.toml`，新增一个指向平台的 provider：

```toml
# 默认模型与 provider
model = "gpt-4o"
model_provider = "bearlabs"

# 第三方 provider 必须关闭响应存储
disable_response_storage = true

[model_providers.bearlabs]
name = "北极熊 AI"
base_url = "https://bearlabs.ai/v1"
env_key = "BEARLABS_API_KEY"
```

然后导出密钥并运行：

```bash
export BEARLABS_API_KEY="YOUR_API_KEY"
codex
```

<Warning>
  `base_url` 用 OpenAI 兼容地址（`/v1`）。第三方 provider 需设 `disable_response_storage = true`，否则部分功能会因官方存储接口不可用而报错。`model` 必须是 [模型接口](/guides/models) 中存在的 ID。
</Warning>
